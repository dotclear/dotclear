Modificar la configuración del tema Blowup te permite personalizarlo muy fácilmente. Para ello, basta con rellenar los campos de configuración o elegir un estilo predefinido.

Estilos predefinidos

You can choose a predefined style in the Predefined styles drop-down list. Once you have chosen the style, you have to submit the form to apply the modifications.

A continuación, puede modificar el estilo predefinido según sus gustos y necesidades.

Colores

Cuando tenga que introducir un valor de color, debe utilizar el formato hexadecimal. Por ejemplo: #FF0000 le dará rojo. También puede utilizar el selector de color situado junto a cada campo de color.

Si te falta inspiración, prueba a visitar las siguientes páginas:

  • Lista de códigos de colores (Wikipedia)
  • Adobe Kuler
  • ColorBlender
  • ColorCombos

Unidades de medida

Cuando tenga que indicar un tamaño, debe ir seguido de una unidad de medida. Por ejemplo: 1em. Si no incluye la unidad, se utilizarán píxeles como unidad de medida por defecto.

Están disponibles las siguientes unidades de medida:

  • px
  • em
  • ex
  • pt
  • %

Imágenes de cabecera

Puede elegir una imagen de cabecera de la lista para sustituir a la predeterminada.

Si elige Custom... en la lista de imágenes, podrá cargar su propia imagen. El archivo de imagen debe estar en formato JPG o PNG y debe tener un ancho exacto de 800px.

Si subes una imagen JPG, se añadirá un borde a la imagen, lo que no ocurre con las imágenes PNG (cuya transparencia también se mantendrá).

Importación y exportación de configuraciones

En la parte inferior de la página, puede visualizar un área denominada Configuration import/export. Esta área de texto contiene la configuración que está utilizando actualmente. Puedes copiarla para compartirla con otras personas.

Para aplicar (importar) una configuración, basta con sustituir el contenido del área de texto por la configuración que desea utilizar. No olvide hacer clic en Aplicar código.
