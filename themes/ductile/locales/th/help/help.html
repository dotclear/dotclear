โดยการปรับเปลี่ยนการตั้งค่าของธีม Ductile คุณสามารถปรับแต่งธีมของคุณได้อย่างง่ายดาย

การปรับเนื้อหาให้เหมาะสมกับบุคคล

ในแท็บนี้คุณสามารถปรับแต่งเนื้อหาของบล็อกของคุณได้

หัวข้อ
ทำเครื่องหมายในช่อง "ซ่อนคำอธิบายของบล็อก" เพื่อไม่ให้แสดงคำอธิบายของบล็อกที่ปกติจะแสดงอยู่ใต้ชื่อเรื่อง

หากปลั๊กอิน "Simple menu" ถูกติดตั้งและเปิดใช้งานแล้ว คุณสามารถคลิกที่ลิงก์ที่กำหนดเพื่อกำหนดค่าเมนูของส่วนหัว (เมนูหลัก?)

ระบุ URL ของโลโก้ที่ต้องการใช้แทนโลโก้เริ่มต้น หาก URL ไม่ได้ขึ้นต้นด้วย/จะถือว่าเป็น URL แบบสัมพัทธ์จากโฟลเดอร์imgของธีม (เช่น การใส่logo.png จะเหมือนกับการใช้โลโก้เริ่มต้น) หากต้องการกลับไปใช้การตั้งค่าเริ่มต้น ให้ล้างข้อมูลในช่องนี้

สติ๊กเกอร์
สติกเกอร์จะแสดงที่ด้านบนของหน้าจอเป็นภาพขนาดเล็กบนหน้าจอขนาดใหญ่ และที่ด้านล่างของหน้าบนหน้าจอขนาดเล็กมาก ภาพทั้งหมดที่มีชื่อไฟล์ตรงกับรูปแบบ sticker-*.png และวางไว้ในโฟลเดอร์ย่อย img ของธีมจะถูกแสดงรายการ คุณสามารถระบุป้ายกำกับและ URL ที่สอดคล้องกันสำหรับแต่ละรายการได้ (หากไม่ต้องการแสดง ให้เว้นว่างทั้งสองช่อง)
คุณยังสามารถปรับเปลี่ยนลำดับการแสดงผลด้วยการลากและวางได้ หากมีการเปิดใช้งาน javascript และหากคุณไม่ได้ปิดการใช้งานในตัวเลือกของคุณ หาก javascript ถูกปิดใช้งาน จะมีช่องให้คุณระบุลำดับการแสดงผล (1 ถึง n)
ประเภทและข้อจำกัดสำหรับรายการโพสต์
บางบริบทการแสดงผลของบล็อกของคุณจะแสดงในรูปแบบของรายการโพสต์ เช่น หน้าแรกหรือหน้าคลังข้อมูลรายเดือน สำหรับแต่ละบริบทที่ระบุในตาราง คุณสามารถกำหนดได้ว่าเนื้อหาของแต่ละโพสต์ที่แสดงควรเป็นเพียงชื่อเรื่อง รูปแบบย่อ (เนื้อหาจากฟิลด์สกัด 300 ตัวอักษรแรก) หรือรูปแบบเต็ม โปรดทราบ: บนหน้าแรก โพสต์แรกจะแสดงในรูปแบบเต็มเสมอ
นอกจากนี้ สำหรับบางบริบทเหล่านั้น คุณยังสามารถระบุจำนวนโพสต์ที่จะแสดงต่อหน้าได้อีกด้วย หากคุณเว้นช่องนี้ไว้ ระบบจะใช้ค่าที่ระบุไว้ในการตั้งค่าทั่วไปของบล็อกแทน

การปรับแต่งการนำเสนอให้เป็นส่วนตัว

ในแท็บนี้คุณสามารถปรับแต่งองค์ประกอบบางอย่างของการนำเสนอของบล็อกของคุณได้

พารามิเตอร์ทั่วไป

พารามิเตอร์เหล่านี้จะถูกใช้กับทุกความกว้างของหน้าจอในบล็อก

ฟอนต์
คุณสามารถเลือกตระกูลตัวอักษรหลักที่นี่ ซึ่งจะถูกใช้สำหรับข้อความส่วนใหญ่ คุณยังสามารถระบุตระกูลตัวอักษรที่สอง (รอง) ซึ่งจะนำมาใช้สำหรับชื่อเรื่องระดับที่สองและคำอธิบายต่างๆ ตัวเลือก "ค่าเริ่มต้น" อนุญาตให้คุณคงการเลือกเดิมไว้
ชื่อบล็อก
คุณสามารถระบุได้ที่นี่ว่าชื่อบล็อกต้องแสดงเป็นตัวหนา ขนาดของมัน (ค่าเริ่มต้นคือ em หน่วยที่รองรับคือ % pt px em และ ex) รวมถึงสี โดยใช้รูปแบบ #rrggbb โดยที่ rr, gg และ bb ต้องเลือกภายใน 0 และ ff
ปล่อยให้ช่องว่างหนึ่งช่องหรือทั้งหมดเพื่อยกเลิกการเลือกของคุณและกลับไปที่ค่าเริ่มต้น
ชื่อตำแหน่ง
คุณสามารถระบุได้ที่นี่ว่าชื่อโพสต์ควรแสดงเป็นตัวหนา ขนาดของชื่อโพสต์ (ค่าเริ่มต้นคือ em หน่วยที่รองรับคือ % pt px em และ ex) รวมถึงสีของชื่อโพสต์โดยใช้ตัวเลือกสีที่มีให้โดยการคลิกที่ไอคอนเลือกสีหรือระบุสีในรูปแบบ #rrggbb โดย rr, gg และ bb ต้องเลือกอยู่ในช่วง 0 ถึง ff
ให้ว่างช่องหนึ่งหรือทุกช่องเพื่อยกเลิกการเลือกของคุณและกลับไปยังตัวเลือกเดิมของคุณ
หัวข้อที่ไม่มีลิงก์
คุณสามารถระบุสีของหัวข้อที่ไม่มีลิงก์ได้ที่นี่ (โดยพื้นฐานแล้ว: ชื่อเรื่องเมื่อแสดงเดี่ยวๆ, หัวข้อย่อยภายในโพสต์, รายการที่ใช้งานอยู่ในเมนูแนวนอน) โดยใช้ตัวเลือกสีที่มีให้โดยการคลิกที่ไอคอนเลือกสีหรือระบุสีในรูปแบบ #rrggbb โดยที่ rr, gg และ bb ต้องเลือกอยู่ในช่วง 0 ถึง ff
ให้ว่างช่องหนึ่งหรือทุกช่องเพื่อยกเลิกการเลือกของคุณและกลับไปค่าเริ่มต้น
ลิงก์ภายในโพสต์
คุณสามารถระบุได้ที่นี่ว่า ลิงก์ที่อยู่ในโพสต์ของคุณควรแสดงเป็นแบบตัวหนาหรือไม่ และกำหนดสีของลิงก์ในสถานะปกติ รวมถึงสถานะ "เคยเข้าชมแล้ว" สีของลิงก์เมื่อใช้งาน หรือเมื่อมีการนำเมาส์ไปวางหรือโฟกัสที่ลิงก์นั้น โดยใช้เครื่องมือเลือกสี (Color Picker) ซึ่งสามารถคลิกที่ไอคอนเลือกสี หรือระบุสีในรูปแบบ #rrggbb โดย rr, gg และ bb ต้องเลือกอยู่ในช่วง 0 ถึง ff
ให้ว่างช่องหนึ่งหรือทุกช่องเพื่อยกเลิกการเลือกของคุณและกลับไปค่าเริ่มต้น

พารามิเตอร์เฉพาะสำหรับโทรศัพท์มือถือ

พารามิเตอร์เหล่านี้จะถูกใช้เฉพาะเมื่อความกว้างของหน้าจอแสดงผลน้อยกว่า 480 พิกเซล (เช่น บนโทรศัพท์มือถือ) หากมีการระบุพารามิเตอร์เหล่านี้ จะใช้แทนพารามิเตอร์ที่เทียบเท่าซึ่งระบุไว้ในพารามิเตอร์ทั่วไปข้างต้น

ชื่อบล็อก
คุณสามารถระบุได้ที่นี่ว่าชื่อบล็อกต้องแสดงเป็นตัวหนา ขนาดของมัน (ค่าเริ่มต้นคือ em หน่วยที่รองรับคือ % pt px em และ ex) รวมถึงสี โดยใช้รูปแบบ #rrggbb โดยที่ rr, gg และ bb ต้องเลือกภายใน 0 และ ff
ปล่อยให้ช่องว่างหนึ่งช่องหรือทั้งหมดเพื่อยกเลิกการเลือกของคุณและกลับไปที่ค่าเริ่มต้น
ชื่อตำแหน่ง
คุณสามารถระบุได้ที่นี่ว่าชื่อโพสต์ควรแสดงเป็นตัวหนา ขนาดของชื่อโพสต์ (ค่าเริ่มต้นคือ em หน่วยที่รองรับคือ % pt px em และ ex) รวมถึงสีของชื่อโพสต์โดยใช้ตัวเลือกสีที่มีให้โดยการคลิกที่ไอคอนเลือกสีหรือระบุสีในรูปแบบ #rrggbb โดย rr, gg และ bb เลือกได้ภายใน 0 ถึง ff
ปล่อยให้ช่องว่างหนึ่งช่องหรือทั้งหมดเพื่อยกเลิกการเลือกของคุณและกลับไปที่ค่าเริ่มต้น

การเข้าถึงและความแตกต่างระหว่างสี

หลังจากเลือกสีแล้ว อัตราส่วนความคมชัดจะแสดงใกล้กับช่องและระบุไว้เมื่อเป็นกรณีนี้ว่าความคมชัดนี้สอดคล้องกับคำแนะนำของ WCAG 2.0 เพื่อให้เป็นไปตามข้อกำหนด ความคมชัดระหว่างสีพื้นหลังและสีตัวอักษรควรมีค่าดังต่อไปนี้:

  • อย่างน้อย 4.5:1 สำหรับข้อความที่มีขนาดปกติสำหรับระดับ AA;
  • อย่างน้อย 7:1 สำหรับข้อความขนาดปกติสำหรับระดับ AAA;
  • อย่างน้อย 3:1 สำหรับข้อความที่มีขนาดขยาย (150% หรือ 120% ตัวหนา) สำหรับระดับ AA;
  • อย่างน้อย 4.5:1 สำหรับข้อความที่มีขนาดขยาย (150% หรือ 120% ตัวหนา) สำหรับระดับ AAA
