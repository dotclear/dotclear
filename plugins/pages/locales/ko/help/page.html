페이지 편집

페이지 제목
페이지 제목을 입력합니다. 이 필드는 필수 입력 사항입니다.
발췌
이 필드는 선택 사항입니다.
콘텐츠
페이지의 콘텐츠입니다. 이 필드는 필수 입력 사항입니다.
참고
이 텍스트 영역은 메모나 메모를 위한 개인적인 용도입니다. 여기에 입력하는 내용은 블로그에 표시되지 않습니다.
페이지 상태
페이지 상태를 선택할 수 있습니다:
  • 보류 중: 게시 상태가 아직 결정되지 않았습니다(아직 게시되지 않음).
  • 예정: 게시일: 필드에 설정된 시간과 날짜에 페이지가 온라인으로 설정됩니다.
  • 게시되지 않았습니다: 페이지가 오프라인 상태입니다.
  • 게시됨: 페이지가 온라인 상태입니다.
게시 날짜 및 시간
여기에서 페이지 게시 날짜와 시간을 변경할 수 있습니다. 페이지 상태가 Scheduled이면 해당 날짜와 시간에 온라인 상태가 됩니다.
페이지 언어
항목의 언어 코드를 선택하세요. 기본값은 사용자의 언어로 설정되어 있지만 원하는 언어 코드(예: "en" 또는 "es-ar")로 변경할 수 있습니다. 이 코드는 항목 표시에 사용됩니다.
텍스트 서식
페이지 구문을 선택합니다. Wiki는 단순화된 구문이며 유효한 HTML로 변환됩니다. HTML을 완벽하게 이해하고 있지 않다면 Wiki 구문을 선택하는 것이 좋습니다. 자세한 내용은 Wiki 구문 참조를 참조하세요.
페이지 위치
이 번호에 따라 페이지를 주문합니다. 이 필드는 페이지 위젯(페이지 위치별 정렬)에서 사용됩니다.
첨부 파일
첨부 파일은 페이지에 추가된 미디어 파일입니다. 이 항목에 파일 추가 버튼을 사용하여 항목에 추가할 미디어 파일을 선택할 수 있습니다.
이 옵션은 항목을 저장한 후에 나타납니다.
자세한 내용은 미디어 관리자 도움말을 확인하세요.
댓글 수락
방문자가 이 페이지에 댓글을 달 수 있도록 허용하려면 이 확인란을 선택합니다. 해당 글로벌 설정은 블로그 설정에서 찾을 수 있습니다.
트랙백 수락
트랙백은 페이지의 일부를 다른 블로그에 댓글로 남기는 방법입니다. 다른 사람들이 귀하의 페이지를 트랙백할 수 있도록 하려면 이 상자를 선택하십시오. 해당 전역 설정은 블로그 설정에서 찾을 수 있습니다.
위젯 페이지에 숨기기
이 옵션을 선택하면 페이지가 활성화되지만 페이지 위젯에 나열되지 않습니다.
비밀번호
페이지의 비밀번호를 입력할 수 있습니다. 비밀번호로 보호된 페이지는 귀하의 블로그에 표시되지 않으며, 귀하가 항목 URL( 사이트에서 이 페이지로 이동 링크 참조)과 비밀번호를 제공한 사람들만 액세스할 수 있습니다. .
기본 이름
잠금을 클릭하면 이 필드의 보호를 해제하고 페이지에 대한 다른 URL을 선택할 수 있습니다. 사용하려는 URL이 이미 다른 페이지에서 사용되고 있는 경우 해당 URL에 숫자가 추가됩니다.

댓글

댓글 목록
탭에서 댓글 페이지의 상태 댓글을 읽고 변경할 수 있습니다. 권한에 따라 편집, 온라인 또는 오프라인 게시, 삭제 또는 바람직하지 않은 댓글로 분류할 수 있습니다.
댓글 추가
탭에서 댓글 추가 블로그를 통하지 않고도 댓글에 직접 추가하거나 답변을 달 수 있습니다. 이 인터페이스를 사용하면 제한 없이 HTML로 의견을 입력할 수도 있습니다. 댓글을 편집할 때와 동일한 방법으로 필드를 채워야 합니다.
