Menu simples

O menu simples permite-lhe apresentar um menu de nível único no seu blogue se o seu tema o suportar.

Lista de itens de menu
Aqui são apresentados os seus itens de menu.
Pode reordenar esta lista arrastando os itens para cima e para baixo.
Também pode modificar a etiqueta, a descrição e o URL de cada item.
Uma vez concluído, clique em Atualizar menu.
Eliminar itens de menu seleccionados
Permite-lhe eliminar itens de menu, assinalando as caixas junto a cada um deles.
Adicionar um novo item de menu
Primeiro, seleccione o tipo de artigo. Em seguida, clique em Continuar…. Depois, se necessário, pode ter de definir um subtipo (categoria, página, etc.). Se for esse o caso, faça-o e clique em Continuar…. Depois, finalmente, pode modificar as propostas feitas para etiqueta, descrição (não obrigatória) e URL do item de menu. Uma vez terminado, clique em Adicionar este item.
  • Etiqueta: O texto âncora. Por exemplo, "Arquivos". Este campo é obrigatório.
  • Descrição: Uma breve descrição para o menu item (será apresentado sob a etiqueta, dependendo do tema selecionado). Por exemplo: "Arquivos de 2004 a 2011".
  • URL: O URL do item. Este campo é obrigatório.
  • Abrir URL num novo separador: Verificar se esta ligação deve ser aberta num novo separador ou numa nova janela.
