Menu simples

O menu simples permite que você exiba um único menu de nível no seu blog, caso seu tema o suporte.

Lista de itens de menu
Aqui são exibidos seus itens de menu.
Você pode reordenar esta lista arrastando os itens para cima e para baixo.
Você também pode modificar a etiqueta, descrição e URL de cada item.
Uma vez feito, clique no menu de atualização.
Excluir itens do menu selecionados
Permite excluir itens do menu marcando as caixas ao lado de cada um deles.
Adicionar um novo item de menu
Primeiro selecione o tipo do item. Em seguida, clique em Continuar…. Então, se necessário, você pode ter que definir um subtipo (categoria, página, etc. . Em caso afirmativo, faça e clique em Continuar…. Então finalmente você pode modificar as propostas feitas para o rótulo , descrição (não obrigatória) e URL do item de menu. Depois disso, clique em Adicionar este item.
  • Rótulo: O texto da âncora. Por exemplo, "Arquivos". Este campo é obrigatório.
  • Descrição: Uma breve descrição para o item (será exibido abaixo da etiqueta dependendo do tema selecionado). Por exemplo: "Arquivos de 2004 a 2011".
  • URL: A URL do item. Este campo é obrigatório.
  • Abrir URL em uma nova aba: Verifique se este link deve ser aberto em uma nova aba ou em uma nova janela.
