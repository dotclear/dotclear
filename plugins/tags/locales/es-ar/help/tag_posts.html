Acciones disponibles en una etiqueta

La primera parte de la página le permite modificar o suprimir una etiqueta.

Cambie el nombre de
Introduzca una nueva palabra para la etiqueta y haga clic en OK. Esto modificará todas las entradas que contengan esta etiqueta.
Borrar esta etiqueta
Este botón eliminará la etiqueta de todas las entradas a las que esté asignada y la borrará de la lista de etiquetas.

Lista de entradas con esta etiqueta

Desde esta lista podrá realizar todas las acciones por lotes que le permitan sus permisos de usuario:

  • Publish: set the entries online.
  • Unpublish: set the entries offline.
  • Schedule: schedule the entries to be set online at their publication date.
  • Mark as pending: erase other publication status.
  • Marcar como seleccionado/no seleccionado: Marca las entradas seleccionadas como seleccionadas/no seleccionadas./li>
  • Change author: allows you to change the posts' author by entering the new author's Id.
  • Delete: suppress the posts (this operation cannot be undone).
  • Cambiar autor: Modifica el autor de las entradas seleccionadas introduciendo el ID del nuevo autor.
  • Borrar: Borrar las entradas seleccionadas (esta operación no se puede cancelar).
  • Añadir/eliminar etiquetas: Añade o elimina etiquetas en las entradas seleccionadas.

La lista anterior puede contener otros elementos si los plugins activos en su sistema han añadido acciones adicionales.
