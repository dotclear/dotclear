Tags

U kunt een tag toekennen aan een item:

  • Door het veld Tags in te vullen. Dit veld heeft een autocomplete functie: voer enkele letters in en het systeem zal bestaande tags voorstellen die deze letters bevatten;
  • Door te klikken op Kies uit lijst. Link om bestaande tags te selecteren.

Tags moeten door komma's worden gescheiden.
