عامل تصفية أكيست

Akismet خدمة فلتر البريد المزعج ذات كفاءة خاصة والتي تتطلب مفتاح API للعمل.

للحصول على مفتاح API، انتقل إلى صفحة تسجيل مفتاح API Akismet واختر الخطة التي تناسبك أكثر من غيرك. يجب عليك ملء النموذج. لا تنسى كتابة إضافات البريد الإلكتروني الخاص بك من أجل تلقي مفتاح API. إذا كان لديك حساب ووردبريس، فيمكنك أيضًا استخدام معرفاتك للحصول على مفتاح API Akismet إذا اخترت الخطة "الشخصية"، سيكون المفتاح مجانًا وستتمكن من استخدامها على مدونة متعددة.

أدخل مفتاح API وحفظه. إذا تم التعرف على المفتاح، يجب أن تحصل على الرسالة التالية "مفتاح API المحدد". العودة إلى قائمة الفلاتر لتفعيل Akismet إذا لم يتم ذلك بعد.
