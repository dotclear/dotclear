Akismet フィルター

Akismetは特に効率的なスパムフィルタサービスであり、動作するためにAPIキーを必要とします。

APIキーを取得するには、 the Akismet API key signup page にアクセスし、最も適したプランを選択してください。フォームに必要事項を記入してください。APIキーを受け取るには、Eメールアドレスを記入することをお忘れなく。Wordpressのアカウントをお持ちの場合は、そのIDを使用してAkismet APIキーを取得することもできます。Personal "プランを選択した場合、APIキーは無料となり、マルチブログで使用することができます。

API キーを入力して保存します。キーが認識された場合は、次のようなメッセージが表示されます。 フィルタリストに戻り、Akismetがまだ終わっていない場合は有効にしてください。
