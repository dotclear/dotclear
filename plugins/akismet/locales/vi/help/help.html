Bộ lọc Akismet

Akismet là một dịch vụ lọc spam đặc biệt hiệu quả, yêu cầu khóa API để hoạt động.

Để lấy khóa API, hãy truy cập trang đăng ký khóa API Akismet và chọn gói dịch vụ phù hợp nhất với bạn. Bạn sẽ cần điền vào một biểu mẫu. Đừng quên ghi lại địa chỉ email của bạn để nhận khóa API. Nếu bạn có tài khoản WordPress, bạn cũng có thể sử dụng thông tin đăng nhập của mình để lấy khóa API Akismet. Nếu bạn chọn gói "Personal", khóa API sẽ miễn phí và bạn có thể sử dụng nó trên nhiều blog.

Nhập khóa API và lưu. Nếu khóa được nhận diện, bạn sẽ nhận được thông báo "Khóa API đã được kiểm tra". Trở lại danh sách bộ lọc để kích hoạt Akismet nếu chưa thực hiện.
