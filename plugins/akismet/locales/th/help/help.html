ตัวกรอง Akismet

Akismet เป็นบริการกรองสแปมที่มีประสิทธิภาพสูงเป็นพิเศษ ซึ่งต้องใช้คีย์ API ในการทำงาน

เพื่อรับกุญแจ API ให้ไปที่หน้าลงทะเบียนกุญแจ API ของ Akismetและเลือกแผนที่เหมาะกับคุณมากที่สุดคุณจะต้องกรอกแบบฟอร์ม อย่าลืมเขียนที่อยู่อีเมลของคุณเพื่อรับคีย์ API หากคุณมีบัญชี Wordpress คุณสามารถใช้ตัวระบุของคุณเพื่อรับคีย์ API ของ Akismet ได้เช่นกัน หากคุณเลือกแผน "ส่วนบุคคล" คีย์จะเป็นฟรี และคุณสามารถใช้ได้กับหลายบล็อก

กรอกคีย์ API และบันทึก หากคีย์ได้รับการยอมรับ คุณจะได้รับข้อความต่อไปนี้ "ตรวจสอบคีย์ API เรียบร้อยแล้ว" กลับไปที่รายการตัวกรองเพื่อเปิดใช้งาน Akismet หากยังไม่ได้ทำ
