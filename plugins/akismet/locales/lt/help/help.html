"Akismet" filtras

"Akismet" yra ypač veiksminga nepageidaujamų laiškų filtravimo paslauga, kurios veikimui reikalingas API raktas.

Norėdami gauti API raktą, eikite į "Akismet" API rakto registracijos puslapį ir pasirinkite jums tinkamiausią planą. Turėsite užpildyti formą. Nepamirškite įrašyti savo el. pašto adreso, kad gautumėte API raktą. Jei turite "Wordpress" paskyrą, norėdami gauti "Akismet" API raktą taip pat galite naudoti savo identifikatorius. Jei pasirinksite planą "Asmeninis", raktas bus nemokamas ir galėsite jį naudoti keliuose tinklaraščiuose.

Įveskite API raktą ir išsaugokite. Jei raktas atpažįstamas, turėtumėte gauti tokį pranešimą "API raktas patikrintas". Grįžkite į filtrų sąrašą ir įjunkite "Akismet", jei tai dar nepadaryta.
