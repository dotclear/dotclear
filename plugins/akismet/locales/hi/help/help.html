अइस्मेट फिल्टर

अइस्मेट एक विशेष रूप से कुशल स्पैम फिल्टर सेवा है जिसमें काम करने के लिए एक एपीआई कुंजी की आवश्यकता होती है।

API कुंजी प्राप्त करने के लिए, Akismet API कुंजी साइनअप पृष्ठ पर जाएँ और वह योजना चुनें जो आपको सबसे अधिक उपयुक्त लगे। आपको एक फ़ॉर्म भरना होगा। API कुंजी प्राप्त करने के लिए अपना ईमेल पता लिखना न भूलें। यदि आपके पास Wordpress खाता है, तो आप Akismet API कुंजी प्राप्त करने के लिए अपने पहचानकर्ताओं का भी उपयोग कर सकते हैं। यदि आप "व्यक्तिगत" योजना चुनते हैं, तो कुंजी मुफ़्त होगी और आप इसे मल्टीब्लॉग पर उपयोग कर पाएंगे।

API कुंजी दर्ज करें और सहेजें. यदि कुंजी को मान्यता प्राप्त है, तो आपको निम्न संदेश "API कुंजी चेक" प्राप्त करना चाहिए. अइस्मेट को सक्रिय करने के लिए फिल्टर सूची पर वापस जाएँ यदि यह अभी तक नहीं किया गया है.
