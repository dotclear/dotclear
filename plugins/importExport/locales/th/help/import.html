การนำเข้าไฟล์แบบแฟลต

นำเข้าบล็อกหรือการติดตั้ง Dotclear แบบเต็มจากไฟล์แบน

บล็อกเดี่ยว

เลือกไฟล์สำรองข้อมูล (ที่ส่งออกจากการติดตั้ง Dotclear อื่น) หรือเลือกไฟล์จากไดเรกทอรีหลักของตัวจัดการสื่อของคุณ แล้วคลิกปุ่มนำเข้าเพื่อเริ่มกระบวนการนำเข้า

หมายเหตุ: เนื้อหาที่นำเข้าในระหว่างกระบวนการนี้จะถูกเพิ่มไปยังเนื้อหาที่มีอยู่แล้วในบล็อกปัจจุบัน ไม่มีสิ่งใดสูญหาย

บล็อกหลายรายการ

เลือกไฟล์สำรองข้อมูลเต็มรูปแบบจากคอมพิวเตอร์ของคุณ หรือเลือกไฟล์จากโฟลเดอร์หลักของตัวจัดการสื่อของคุณ ป้อนรหัสผ่านของคุณ และคลิกที่ปุ่มนำเข้าเพื่อเริ่มกระบวนการนำเข้าเต็มรูปแบบ

คำเตือน:เนื้อหาทั้งหมดของบล็อกปัจจุบันจะถูกลบและแทนที่ด้วยเนื้อหาจากไฟล์สำรองข้อมูล ผู้ใช้ปัจจุบันเท่านั้นที่จะถูกเก็บไว้ ผู้ใช้เพิ่มเติมจากไฟล์สำรองข้อมูลจะถูกเพิ่มเข้าไปในรายชื่อผู้ใช้ปัจจุบัน

นำเข้าฟีด RSS หรือ Atom

เพิ่มเนื้อหาฟีดลงในบล็อกปัจจุบัน

กรุณากรอก URL ของ RSS หรือ Atom feed ที่คุณต้องการนำเข้า และคลิกที่นำเข้า. เนื้อหาที่พบใน feed จะถูกเพิ่มเข้าไปในเนื้อหาบล็อกปัจจุบันของคุณ.

การนำเข้า Dotclear 1.2

คุณสมบัตินี้ช่วยให้คุณสามารถนำเข้าการติดตั้ง Dotclear 1.2 ไปยังบล็อกปัจจุบันของคุณได้ กระบวนการนี้จะแทนที่หมวดหมู่, บล็อกโรล, รายการ และความคิดเห็นของบล็อกปัจจุบัน

ป้อน URL ของเซิร์ฟเวอร์ที่ฐานข้อมูล Dotclear 1.2 ที่จะนำเข้าตั้งอยู่ จากนั้นป้อนชื่อฐานข้อมูล ชื่อผู้ใช้ และรหัสผ่านเพื่อเชื่อมต่อกับฐานข้อมูล และคำนำหน้าตารางฐานข้อมูลที่ใช้ในการติดตั้ง Dotclear 1.2 คุณสามารถหาข้อมูลนี้ได้ในไฟล์conf/config.phpจากการติดตั้ง Dotclear 1.2 ของคุณ

คลิกที่ปุ่มนำเข้าบล็อกของฉันตอนนี้เพื่อเริ่มกระบวนการนำเข้า

คุณสามารถลดหรือเพิ่มจำนวนรายการที่นำเข้าพร้อมกันได้โดยใช้ช่องจำนวนรายการที่จะนำเข้าพร้อมกันขึ้นอยู่กับความเร็วในการเข้าถึงของการติดตั้ง Dotclear 1.2 ของคุณ

นำเข้าเวิร์ดเพรส

คุณลักษณะนี้ช่วยให้คุณสามารถนำเข้าการติดตั้ง Wordpress ไปยังบล็อกปัจจุบันของคุณได้ กระบวนการนี้จะแทนที่หมวดหมู่, บล็อกโรล, รายการ และความคิดเห็นของบล็อกปัจจุบัน

ป้อน URL ของเซิร์ฟเวอร์ที่ฐานข้อมูล Wordpress ที่จะนำเข้าตั้งอยู่ จากนั้นป้อนชื่อฐานข้อมูล ชื่อผู้ใช้ และรหัสผ่านเพื่อเชื่อมต่อกับฐานข้อมูล และคำนำหน้าตารางฐานข้อมูลที่ใช้ในการติดตั้ง Wordpress

คุณสามารถตั้งค่าตัวเลือกการนำเข้าได้ไม่กี่อย่างเกี่ยวกับการนำเข้าหมวดหมู่และจำนวนรายการที่จะนำเข้าในคราวเดียว คุณยังสามารถกำหนดไวยากรณ์ที่ใช้สำหรับรายการและเนื้อหาของความคิดเห็น (HTML, Wiki หรืออื่น ๆ ขึ้นอยู่กับไวยากรณ์ที่ติดตั้งและเปิดใช้งานอยู่ในปัจจุบัน)

คลิกที่นำเข้าบล็อกของฉันตอนนี้เพื่อเริ่มกระบวนการนำเข้า
