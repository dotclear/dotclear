Import av platta filer

Importerar en blogg eller en fullständig Dotclear-installation från en platt fil.

Enskild blogg

Välj säkerhetskopian (exporterad från en annan Dotclear-installation) eller välj en fil från rotkatalogen i din mediahanterare och klicka på knappen Import för att starta importprocessen.

Observera: det innehåll som importeras under denna process kommer att läggas till det befintliga innehållet i den aktuella bloggen, ingenting kommer att gå förlorat.

Flera bloggar

Välj den fullständiga säkerhetskopian från din dator eller välj en från rotmappen i din mediahanterare, ange ditt lösenord och klicka på knappen Import för att starta den fullständiga importprocessen.

Varning: hela innehållet i den aktuella bloggen kommer att raderas och ersättas av innehållet i säkerhetskopian. Endast nuvarande användare kommer att behållas. Ytterligare användare från säkerhetskopian kommer att läggas till i listan över nuvarande användare.

Import av RSS- eller Atom-flöden

Lägg till ett feed-innehåll till den aktuella bloggen.

Ange webbadressen till det RSS- eller Atom-flöde som du vill importera och klicka på Import. Det innehåll som finns i flödet kommer att läggas till i ditt aktuella blogginnehåll.

Import av Dotclear 1.2

Med den här funktionen kan du importera en Dotclear 1.2-installation till din nuvarande blogg. Denna process kommer att ersätta kategorier, blogrolls, poster och kommentarer i den aktuella bloggen.

Ange serverns URL där Dotclear 1.2-databasen som ska importeras finns, ange sedan databasens namn, användarnamn och lösenord för att ansluta till databasen och prefixet för de databastabeller som används i Dotclear 1.2-installationen. Du hittar denna information i filen conf/config.php från din Dotclear 1.2-installation.

Klicka på knappen Importera min blogg nu för att starta importprocessen.

Du kan minska eller öka antalet poster som importeras på en gång med hjälp av fältet Number of entries to import at once , beroende på åtkomsthastigheten för din Dotclear 1.2-installation.

Wordpress-import

Med den här funktionen kan du importera en Wordpress-installation till din nuvarande blogg. Denna process kommer att ersätta kategorier, bloggrullar, poster och kommentarer i den aktuella bloggen.

Ange serveradressen där Wordpress-databasen som ska importeras finns, ange sedan databasnamnet, användarnamnet och lösenordet för att ansluta till databasen samt prefixet för de databastabeller som används i Wordpress-installationen.

Du kan ställa in några importalternativ för import av kategorier och antalet poster som ska importeras på en gång. Du kan också definiera vilken syntax som ska användas för posterna och kommentarerna (HTML, Wiki eller andra beroende på vilka syntaxer som är installerade och aktiverade).

Klicka på Importera min blogg nu för att starta importprocessen.

Observera: exportfunktionerna finns nu på sidanMaintenance .
