Importazione file piatto

Importa un blog o un'installazione completa Dotclear da un file piatto.

Singolo blog

Selezionare il file di backup (esportato da un'altra installazione Dotcleare) o selezionarne uno dalla directory principale del media manager e fare clic sul pulsante Importa per avviare il processo di importazione.

Nota: il contenuto importato durante questo processo sarà aggiunto al contenuto esistente del blog corrente, niente sarà perso.

Blog multipli

Selezionare il file di backup completo dal computer o selezionarne uno dalla cartella principale del vostro media manager, inserisci la tua password e clicca sul pulsante Importa per avviare il processo di importazione completo.

Attenzione: l'intero contenuto del blog corrente verrà eliminato e sostituito dal contenuto del file di backup. Verranno mantenuti solo gli utenti correnti. Ulteriori utenti dal file di backup verranno aggiunti alla lista degli utenti correnti.

Importazione feed RSS o Atom

Aggiungi un contenuto feed al blog corrente.

Inserisci l'URL del feed RSS o Atom che vuoi importare e clicca su Importa. Il contenuto trovato nel feed verrà aggiunto al contenuto del blog attuale.

Dotclear 1.2 import

Questa funzione consente di importare un'installazione Dotclear 1.2 nel tuo blog attuale. Questo processo sostituirà le categorie, blogroll, voci e commenti del blog attuale.

Inserisci l'URL del server dove il Dotclear 1. il database da importare è localizzato, quindi inserisci il nome del database, il nome utente e la password per connettersi al database e al prefisso delle tabelle del database utilizzato nel Dotclear 1. installazione. Puoi trovare queste informazioni nel file conf/config.php dalla tua installazione Dotclear 1.2.

Clicca sul pulsante Importa ora il mio blog per avviare il processo di importazione.

Puoi diminuire o aumentare la quantità di voci importate contemporaneamente utilizzando il Numero di voci da importare contemporaneamente , a seconda della velocità di accesso del tuo Dotclear 1. installazione.

Importazione Wordpress

Questa funzione consente di importare un'installazione di Wordpress nel tuo blog attuale. Questo processo sostituirà le categorie, blogroll, voci e commenti del blog attuale.

Inserisci l'URL del server dove si trova il database Wordpress da importare, quindi inserisci il nome del database, il nome utente e la password per connettersi al database e il prefisso tabelle del database utilizzato nell'installazione di Wordpress.

È possibile impostare alcune opzioni di importazione per quanto riguarda l'importazione delle categorie e il numero di voci da importare contemporaneamente. È inoltre possibile definire la sintassi utilizzata per le voci e il contenuto dei commenti (HTML, Wiki o altri a seconda delle sintassi attualmente installate e attive).

Clicca su Importa ora il mio blog per avviare il processo di importazione.

Nota: le funzioni di esportazione si trovano ora nella paginaManutenzione.
