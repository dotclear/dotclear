フラットファイルのインポート

フラットファイルからブログやDotclearの完全インストールをインポートします。

単一ブログ

バックアップ ファイル(別の Dotclear インストールからエクスポート) を選択するか、メディア マネージャのルート ディレクトリから選択して インポート ボタンをクリックしてインポート処理を開始します。

注意: このプロセス中にインポートされたコンテンツは、現在のブログの既存のコンテンツに追加されます。失われることはありません。

複数のブログ

フルバックアップファイルをコンピュータから選択するか、メディア・マネージャのルートフォルダから選択してください。 パスワードを入力し、 Import ボタンをクリックして、完全なインポート処理を開始します。

警告: 現在のブログの内容全体が削除され、バックアップファイルの内容に置き換えられます。 現在のユーザーのみが保持されます。バックアップファイルの追加ユーザーは現在のユーザーリストに追加されます。

RSS または Atom フィードのインポート

フィードコンテンツを現在のブログに追加します。

インポートするRSSまたはAtomフィードのURLを入力し、 Importをクリックします。 フィードに含まれるコンテンツは、現在のブログコンテンツに追加されます。

Dotclear 1.2 のインポート

この機能を使用すると、Dotclear 1.2のインストールを現在のブログにインポートできます。 このプロセスは、現在のブログのカテゴリ、ブログロール、エントリ、コメントを置き換えます。

Dotclear 1のサーバーURLを入力します。 データベースが検索され、データベース名を入力します。 データベースに接続するためのユーザー名とパスワードと、Dotclear 1で使用されるデータベーステーブルのプレフィックス。 installation. Dotclear 1.2の conf/config.php ファイルでこの情報を見つけることができます。

今すぐブログをインポート ボタンをクリックしてインポート処理を開始します。

Number of entries to import at once field を使用して、一度にインポートされたエントリの数を減らしたり増やしたりできます。 Dotclear 1のアクセス速度に応じて。 インスタレーションです

Wordpress import

この機能は、現在のブログにWordpressのインストールをインポートすることができます。 このプロセスは、現在のブログのカテゴリ、ブログロール、エントリ、コメントを置き換えます。

Wordpressのデータベースが存在するサーバーURLを入力し、データベース名を入力します。 データベースに接続するためのユーザー名とパスワードと、Wordpressのインストールで使用されるデータベーステーブルのプレフィックス。

カテゴリのインポートと一度にインポートする項目の数に関するいくつかのインポートオプションを設定できます。 エントリとコメントのコンテンツ (HTML) に使用される構文を定義することもできます。 現在インストールされている構文と有効化された構文に応じて、Wiki などがあります。

今すぐブログをインポート をクリックしてインポート処理を開始します。

注: エクスポート関数はメンテナンス ページにあります。
