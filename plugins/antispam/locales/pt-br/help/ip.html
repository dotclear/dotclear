Filtro IP

Este filtro permite-lhe colocar manualmente na lista negra ou na lista branca os endereços IP. Um comentário proveniente de um endereço IP listado na secção da lista negra será imediatamente considerado como lixo. Um comentário proveniente de um endereço IP listado na secção da lista branca será considerado fiável e os outros filtros antispam não o processarão.

Para adicionar um IP, primeiro selecione a guia desejada (Blacklist ou Whitelist) e digite o endereço IP. A caixa de seleção Global IP está disponível se você for um super administrador e permitirá que você adicione o endereço IP a todos os blogs da sua instalação Dotclear.

É possível adicionar endereços IP com uma máscara de rede para bloquear ou permitir um intervalo de endereços IP. Por exemplo:

  • 127.0.0.0/255.0.0.0 ou a forma abreviada 127/8 reconhecerá todos os endereços IP como 127.xxx.xxx.xxx
  • 127.0.0.0/255.255.0.0 ou a forma abreviada 127.0/16 reconhecerá todos os endereços IP como 127.0.xxx.xxx
  • 127.0.0.0/255.255.255.0 ou a forma abreviada 127.0.0/24 reconhecerá todos os endereços IP como 127.0.0.xxx
