ตัวกรอง IP

ตัวกรองนี้ช่วยให้คุณสามารถเพิ่มที่อยู่ IP ลงในบัญชีดำหรือบัญชีขาวได้ด้วยตนเอง ความคิดเห็นที่มาจากที่อยู่ IP ที่อยู่ในส่วนบัญชีดำจะถูกพิจารณาเป็นขยะทันที ความคิดเห็นที่มาจากที่อยู่ IP ที่อยู่ในส่วนบัญชีขาวจะถูกพิจารณาว่าเป็นที่เชื่อถือได้ และตัวกรองป้องกันสแปมอื่น ๆ จะไม่ประมวลผลความคิดเห็นนั้น

เพื่อเพิ่ม IP ให้เลือกแท็บที่คุณต้องการก่อน (BlacklistหรือWhitelist) และป้อนที่อยู่ IP ที่ต้องการ. กล่องกาเครื่องหมายGlobal IPจะปรากฏให้คุณเลือกหากคุณเป็นผู้ดูแลระบบระดับสูง และจะอนุญาตให้คุณเพิ่มที่อยู่ IP ไปยังบล็อกทั้งหมดในระบบ Dotclear ของคุณ.

คุณสามารถเพิ่มที่อยู่ IP พร้อมมาสก์เครือข่ายเพื่อบล็อกหรืออนุญาตช่วงของที่อยู่ IP ได้ ตัวอย่างเช่น:

  • 127.0.0.0/255.0.0.0หรือรูปแบบย่อ127/8จะรับรู้ที่อยู่ IP ทั้งหมด เช่น127.xxx.xxx.xxx
  • 127.0.0.0/255.255.0.0หรือรูปแบบย่อ127.0/16จะรู้จักที่อยู่ IP ทั้งหมดเช่น127.0.xxx.xxx
  • 127.0.0.0/255.255.255.0หรือรูปแบบย่อ127.0.0/24จะรับรู้ที่อยู่ IP ทั้งหมด เช่น127.0.0.xxx
