Κακές Λέξεις

Αυτό θα φιλτράρει τα σχόλια με βάση τις λέξεις που χρησιμοποιούνται στο όνομα χρήστη, διεύθυνση ηλεκτρονικού ταχυδρομείου, διεύθυνση URL ιστοσελίδας ή το περιεχόμενο σχολίου.

Προσθήκη λέξης
Ομοίως με το φιλτράρισμα διευθύνσεων IP, απλά προσθέστε τη λέξη που θα θέλατε να φιλτράρετε. Μπορείτε επίσης, αν έχετε τα κατάλληλα δικαιώματα, να ορίσετε τις καθολικές λέξεις που θα χρησιμοποιηθούν για όλα τα blogs στην εγκατάσταση Dotclear σας.
Υπόδειξη: Μπορείτε να προσθέσετε μια κανονική έκφραση στη λίστα λέξεων σας προσθέτοντας ένα / στην αρχή και στο τέλος της λέξης. Για παράδειγμα, το /foo/ θα φιλτράρει όλες τις λέξεις που περιέχουν foo.
Λίστα κακών λέξεων
Λίστα λέξεων που απαριθμούν το σχόλιο ως σκουπίδια.
Για να αφαιρέσετε λέξεις από αυτή τη λίστα, επιλέξτε τη λέξη(ες) και πατήστε Διαγραφή επιλεγμένων λέξεων.
