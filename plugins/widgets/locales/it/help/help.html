Uso di widget

I widget possono essere utilizzati per aggiungere vari blocchi di contenuti da visualizzare sulle pagine pubbliche. Per aggiungere un widget, trascinalo dalla lista Oggetti disponibili a sinistra su una delle barre laterali a destra di questa pagina. Puoi ordinare i tuoi widget in una barra laterale trascinandoli su o giù. Per applicare le modifiche, premi Aggiorna barre laterali.

Una volta inclusi in una barra laterale, i widget hanno opzioni di configurazione che puoi raggiungere facendo clic sulla freccia accanto al loro nome.

Per rimuovere un widget, basta trascinarlo in una delle zone dedicate

Ricordati di salvare le modifiche dopo ogni modifica premendo Aggiorna le barre laterali

Usa il pulsante Reimposta le barre laterali per reimpostare le barre laterali come originariamente impostate nel tuo sistema (un pop-up ti chiederà di confermare la tua scelta).

Tag Widget

Se hai il permesso di modificare i tuoi modelli di tema, puoi aggiungere direttamente widget come template tag, con la loro configurazione.

Per aggiungere un widget nel tuo modello, devi scrivere un codice in questo modo:

<tpl:Widget id="Widget ID">
  <setting name="Setting name">Impostazione del valore</setting>
  ...
</tpl:Widget>
