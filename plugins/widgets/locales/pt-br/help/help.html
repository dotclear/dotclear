Utilização de widgets

Os widgets podem ser utilizados para adicionar vários blocos de conteúdo a apresentar nas suas páginas públicas. Para adicionar um widget, arraste-o da lista Widgets disponíveis à esquerda para uma das barras laterais à direita desta página. Pode ordenar os widgets numa barra lateral arrastando-os para cima ou para baixo. Para aplicar as alterações, prima Atualizar barras laterais.

Uma vez incluídos numa barra lateral, os widgets têm opções de configuração que podem ser acedidas clicando na seta junto ao seu nome.

Para remover um widget, basta arrastá-lo para uma das zonas dedicadas

Não se esqueça de guardar as suas alterações após qualquer modificação, premindo Atualizar barras laterais

Utilize o botão Reset sidebars para repor as barras laterais tal como foram originalmente definidas no seu sistema (uma janela pop-up pedir-lhe-á para confirmar a sua escolha).

Etiquetas de widgets

Se tiver permissão para editar os seus modelos de tema, pode adicionar diretamente widgets como etiquetas de modelos, com a sua própria configuração.

Para adicionar um widget ao seu modelo, tem de escrever um código como este:

<tpl:Widget id="Widget ID">
  <setting name="Setting name">Valor de definição</setting>
  ...
</tpl:Widget>
