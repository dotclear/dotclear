Edytor motywów

Edytor motywów umożliwia modyfikowanie plików szablonów, arkuszy stylów i plików Javascript aktualnie używanego motywu.

Lista plików po prawej stronie jest podzielona na pięć sekcji:

  • Pliki szablonów
  • Pliki CSS (arkusze stylów)
  • Pliki Javascript
  • Pliki Locales (pliki używane do tłumaczenia)
  • Pliki PHP (należy je modyfikować z zachowaniem ostrożności)

Każda sekcja plików może mieć cztery podsekcje:

  • Pliki niestandardowe
  • Pliki należące do motywu
  • Pliki należące do motywu nadrzędnego
  • Pliki należące do zestawu szablonów

Mofidacja plików

Aby zmodyfikować plik, kliknij jego nazwę. Jego zawartość zostanie wyświetlona w edytorze. Jeśli plik można nadpisać, można go zapisać, naciskając przycisk Save (s).

Zachowaj ostrożność: Modyfikacje są stosowane natychmiast po zapisaniu.

Jeśli zmodyfikujesz plik pochodzący z zestawu szablonów lub motywu nadrzędnego, zostanie on skopiowany do bieżącego motywu.

Podświetlanie składni

Podświetlanie składni można aktywować w preferencjach (w zakładce moje opcje).

Więcej informacji

Aby edytować pliki szablonów, sprawdź listę tagów szablonów.
