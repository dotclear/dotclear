থিম সম্পাদক

থিম সম্পাদক আপনাকে টেমপ্লেট ফাইল, stylesheets এবং আপনি বর্তমানে ব্যবহার করা থিমের জাভাস্ক্রিপ্ট ফাইল পরিবর্তন করতে পারবেন.

ডানদিকের ফাইল তালিকাটি পাঁচটি বিভাগে বিভক্ত:

  • টেমপ্লেট ফাইল
  • CSS ফাইল (stylesheets)
  • জাভাস্ক্রিপ্ট ফাইল
  • লোকেশন ফাইল (অনুবাদের জন্য ব্যবহৃত ফাইল)
  • পিএইচপি ফাইল (সাবধানতার সাথে পরিবর্তন করতে হবে)

প্রতিটি ফাইলের বিভাগে চারটি উপ-বিভাগ থাকতে পারে:

  • কাস্টমাইজড ফাইল
  • থিমের সাথে সম্পর্কিত ফাইলগুলি
  • প্যারেন্ট থিম এর ফাইলগুলি
  • টেম্পলেট সেটের ফাইলগুলো

ফাইল mofidication

একটি ফাইল সংশোধন করতে, তার নামের উপর ক্লিক করুন। এর বিষয়বস্তু সম্পাদকটিতে প্রদর্শিত হবে। যদি ফাইলটি ওভাররাইট করা যায় তবে আপনি সেভ (গুলি) বোতাম টিপে এটি সংরক্ষণ করতে সক্ষম হবেন।

সতর্ক থাকুন: সংরক্ষণ করার পরপরই পরিবর্তন আবেদন করা হয়.

যদি আপনি একটি ফাইল পরিবর্তন করেন যা টেম্পেট সেট অথবা প্যারেন্ট থিম থেকে আসে, তবে ফাইলটি আপনার বর্তমান থিম-এ কপি করা হবে.

সিনট্যাক্স হাইলাইট

আপনি আপনার পছন্দ ( আমার বিকল্প ট্যাবে) সিনট্যাক্স হাইলাইটিং সক্রিয় করতে পারেন।

আরো পড়া

আপনার টেমপ্লেট ফাইলগুলি সম্পাদনা করতে, দয়া করে টেমপ্লেট ট্যাগগুলির তালিকা দেখুন।
