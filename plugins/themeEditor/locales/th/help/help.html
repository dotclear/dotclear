ตัวแก้ไขธีม

ตัวแก้ไขธีมช่วยให้คุณสามารถแก้ไขไฟล์แม่แบบ, แผ่นสไตล์ และไฟล์ Javascript ของธีมที่คุณกำลังใช้งานอยู่ได้

รายการไฟล์ทางด้านขวาแบ่งออกเป็นห้าส่วน:

  • ไฟล์แม่แบบ
  • ไฟล์ CSS (สไตล์ชีต)
  • ไฟล์ Javascript
  • ไฟล์โลคัล (ไฟล์ที่ใช้สำหรับการแปล)
  • ไฟล์ PHP (ต้องแก้ไขด้วยความระมัดระวัง)

แต่ละส่วนของไฟล์อาจมีสี่ส่วนย่อย:

  • ไฟล์ที่ปรับแต่งตามความต้องการ
  • ไฟล์ที่เป็นของธีม
  • ไฟล์ที่เป็นของธีมหลัก
  • ไฟล์ที่เป็นของชุดแม่แบบ

การแก้ไขไฟล์

หากต้องการแก้ไขไฟล์ ให้คลิกที่ชื่อไฟล์นั้น เนื้อหาของไฟล์จะแสดงในตัวแก้ไข หากคุณสามารถเขียนทับไฟล์ได้ คุณสามารถบันทึกไฟล์ได้โดยกดปุ่มบันทึก (s)

โปรดระวัง: การแก้ไขจะถูกนำไปใช้ทันทีหลังจากบันทึก

หากคุณแก้ไขไฟล์ที่มาจากชุดเทมเพลตหรือธีมหลัก ไฟล์นั้นจะถูกคัดลอกไปยังธีมปัจจุบันของคุณ

การเน้นไวยากรณ์

คุณสามารถเปิดใช้งานการเน้นไวยากรณ์ได้ในตัวเลือกของคุณ(ในแท็บตัวเลือกของฉัน)

อ่านเพิ่มเติม

เพื่อแก้ไขไฟล์เทมเพลตของคุณ โปรดตรวจสอบรายการแท็กเทมเพลต
