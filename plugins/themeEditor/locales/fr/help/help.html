Éditeur de thème

L'éditeur de thème vous permet de modifier les fichiers de modèle, les feuilles de style et les fichiers Javascript du thème que vous utilisez actuellement.

La liste de fichiers à droite est divisée en quatre sections :

  • fichiers du thème : les templates (du thème et du jeu de template),
  • fichiers CSS : les feuilles de style,
  • fichiers JavaScript.
  • fichiers de traduction.
  • Fichiers PHP (à modifier avec précaution)

Chaque section de fichiers peut comporter trois sous-sections :

  • jaune indique qu'il fait partie du thème actif.
  • jaune indique qu'il fait partie du thème actif.
  • Fichiers appartenant à l'ensemble de modèles
  • rouge indique qu'il fait partie d'un thème parent.

Modification du fichier

A noter que si vous modifiez un fichier se trouvant dans le thème "default", celui-ci sera copié dans votre thème en cours d'utilisation.

Notez que vous pouvez activer la coloration syntaxique dans vos préférences (onglet « Mes options »).

Si vous modifiez un fichier provenant du jeu de modèles ou du thème parent, le fichier sera copié dans votre thème actuel.

Coloration syntaxique

Vous pouvez activer la coloration syntaxique dans vos préférences (onglet Mes options).
