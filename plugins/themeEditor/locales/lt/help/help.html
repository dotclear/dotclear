Temos redaktorius

Temos redaktorius leidžia keisti šiuo metu naudojamos temos šablono failus, stilių rinkmenas ir Javascript failus.

Dešinėje pusėje esantis failų sąrašas suskirstytas į keturis skyrius:

  • Šablonų failai
  • CSS failai (stilių rinkmenos)
  • Javascript failai
  • Vietovių failai (vertimui naudojami failai)
  • PHP failai (modifikuoti atsargiai)

Kiekvienas failų skyrius gali turėti tris poskyrius:

  • Dabartinei temai priklausantys failai
  • Pagrindinei temai priklausantys failai
  • Šablono rinkiniui priklausantys failai

Failų mofidikacija

Jei norite keisti failą, spustelėkite jo pavadinimą. Jo turinys bus rodomas redaktoriuje. Jei failą galima perrašyti, galėsite jį išsaugoti paspaudę mygtuką Save (s).

Būkite atsargūs: Pakeitimai pritaikomi iš karto po išsaugojimo.

Jei pakeisite failą iš šablonų rinkinio arba pagrindinės temos, jis bus nukopijuotas į dabartinę temą.

Sintaksės paryškinimas

Sintaksės paryškinimą galite įjungti nustatymuose (skirtuke Mano parinktys).

Tolesnis skaitymas

Norėdami redaguoti šablono failus, peržiūrėkite šablonų žymių sąrašą.
