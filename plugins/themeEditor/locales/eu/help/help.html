Gaien editorea

Gaien editoreak txantiloi-fitxategiak, estilo-orriak eta une honetan erabiltzen ari zaren gaiaren Javascript fitxategiak alda ditzakezu.

Eskuineko fitxategien zerrenda lau ataletan banatuta dago:

  • Txantiloi fitxategiak
  • CSS fitxategiak (estilo-orriak)
  • Javascript fitxategiak
  • Tokiko fitxategiak (itzultzeko erabiltzen diren fitxategiak)

Fitxategien atal bakoitzak hiru azpiatal izan ditzake:

  • Uneko gaiari dagozkion fitxategiak
  • Guraso gaiari dagozkion fitxategiak
  • Txantiloi multzoari dagozkion fitxategiak

Fitxategi bakoitzaren aurrean buletak hiru kolore ezberdin izan ditzake:

  • Horiak esan nahi du uneko gaiarena dela.
  • Gorriz esan nahi du guraso-gai batena dela.
  • Beltzak txantiloi multzoan dagoela esan nahi du.

Uneko gaiari dagozkion fitxategiak ere lodiz daude.

Fitxategien modifikazioa

Fitxategi bat aldatzeko, egin klik haren izenan. Bere edukia editorean bistaratuko da. Fitxategia gainidazten bada, gorde ahal izango duzu Gorde(k) botoia sakatuz.

Kontuz: aldaketak gorde eta berehala aplikatzen dira.

Txantiloi multzotik edo gai nagusitik datorren fitxategi bat aldatzen baduzu, fitxategia zure uneko gaian kopiatuko da.

Sintaxia nabarmentzea

Sintaxia nabarmentzea zure hobespenetan aktiba dezakezu ( Nire aukerak fitxan).

Irakurketa gehiago

Txantiloi-fitxategiak editatzeko, begiratu txantiloi-etiketen zerrenda.
