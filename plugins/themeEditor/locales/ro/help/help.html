Editor de teme

Editorul de teme vă permite să modificați fișierele șablon, foile de stil și fișierele Javascript ale temei pe care o utilizați în prezent.

Lista de fișiere din dreapta este împărțită în patru secțiuni:

  • Fișiere șablon
  • Fișiere CSS (foi de stil)
  • Fișiere Javascript
  • Fișiere locale (fișiere utilizate pentru traducere)
  • Fișiere PHP (a se modifica cu precauție)

Fiecare secțiune de fișiere poate avea trei subsecțiuni :

  • Fișiere aparținând temei părinte
  • Fișiere aparținând temei părinte
  • Fișiere aparținând setului de șabloane
  • Fișiere aparținând setului de șabloane

Mofidicarea fișierelor

Pentru a modifica un fișier, faceți clic pe numele acestuia. Conținutul acestuia va fi afișat în editor. În cazul în care fișierul poate fi suprascris, îl veți putea salva apăsând butonul Save (s).

Aveți grijă: Modificările sunt aplicate imediat după salvare.

Dacă modificați un fișier care provine din setul de șabloane sau din tema părinte, fișierul va fi copiat în tema curentă.

Evidențierea sintaxei

Puteți activa evidențierea sintaxei în preferințele dvs. (în fila Opțiunile mele).

Lecturi suplimentare
