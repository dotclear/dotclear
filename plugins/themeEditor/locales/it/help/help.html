Editor Tema

L'editor dei temi consente di modificare i file dei modelli, i fogli di stile e i file Javascript del tema che stai utilizzando.

L'elenco dei file a destra è diviso in quattro sezioni:

  • File modello
  • File CSS (fogli di stile)
  • File Javascript
  • File di localizzazione (file usati per la traduzione)

Ogni sezione di file può avere tre sottosezioni:

  • File appartenenti al tema corrente
  • File appartenenti al tema genitore
  • File appartenenti al modello impostato

Il proiettile di fronte ad ogni file può avere tre colori diversi:

  • Giallo significa che appartiene al tema corrente.
  • Rosso significa che appartiene a un tema genitore.
  • Nero significa che è nel set di modelli.

Anche i file appartenenti al tema corrente sono in grassetto.

Mofidication file

Per modificare un file, fare clic sul suo nome. Il suo contenuto verrà visualizzato nell'editor. Se il file può essere sovrascritto, potrai salvarlo premendo il pulsante Salva (s).

Attenzione: le modifiche vengono applicate immediatamente dopo il salvataggio.

Se si modifica un file che viene dal modello impostato o dal tema padre, il file verrà copiato nel tema corrente.

Evidenziazione della sintassi

Puoi attivare l'evidenziazione della sintassi in le tue preferenze (nella scheda Le mie opzioni).

Ulteriore lettura

Per modificare i file del modello, controlla l'elenco dei tag del modello.
