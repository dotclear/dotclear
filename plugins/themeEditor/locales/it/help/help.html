Editor Tema

L'editor del tema consente di modificare i file dei modelli, i fogli di stile, i file Javascript, i locali e alcuni file PHP del tema in uso.

L'elenco dei file sulla destra è suddiviso in cinque sezioni:

  • File modello
  • File CSS (fogli di stile)
  • File Javascript
  • File di localizzazione (file usati per la traduzione)
  • File PHP (da modificare con cautela)

Ogni sezione di file può avere quattro sottosezioni:

  • File personalizzati
  • File appartenenti al tema corrente
  • File appartenenti al tema genitore
  • File appartenenti al modello impostato

Modifica dei file

Per modificare un file, fare clic sul suo nome. Il suo contenuto verrà visualizzato nell'editor. Se il file può essere sovrascritto, potrai salvarlo premendo il pulsante Salva (s).

Attenzione: le modifiche vengono applicate immediatamente dopo il salvataggio.

Per ripristinare le modifiche di un file, fare clic sul pulsante di ripristino. Il file personalizzato verrà quindi rimosso.

Evidenziazione della sintassi

È possibile attivare l'evidenzi azione della sintassi nelle preferenze (scheda Opzioni).

Ulteriore lettura

Per modificare i file del modello, controlla l'elenco dei tag del modello.
