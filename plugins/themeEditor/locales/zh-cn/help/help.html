主题编辑器

主题编辑器让您改变正在使用的主题的模板文件，样式表和 JavaScript 文件。

文件列表被划分为三个部分 ：

  • 模板文件：所有模板
  • CSS 文件：所有样式表
  • JavaScript 文件
  • 地域文件（用于翻译的文件）。

文件名前面的黄色图标表明这个文件是当前主题的一部分。黑色图标意味着这是默认主题的一部分。

  • 属于当前主题的文件
  • 属于父主题的文件
  • 属于模板集的文件

如果您修改一个属于默认主题的文件，它将会被复制到您正在使用的主题里面。

  • 黄色 ，意味着它属于当前的主题。
  • 红色 ，意味着它属于一个父主题。
  • 黑色 ，意味着它在模板集中。

要修改一个文件，点击它的名字以将它的内容显示在编辑区域。如果文件是可写的，您将可以通过点击“保存”来保存您的修改。

文件的处理

修改会在保存后立刻应用，所以要小心。

为了帮助修改您的模板，您可以参考模板标签列表。

如果你修改一个来自模板集或父主题的文件，该文件将被复制到你当前的主题中。

语法高亮

你可以在 你的首选项 （在 我的选项 标签）中激活语法高亮。
