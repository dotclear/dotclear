עורך ערכה

עורך הערכות מאפשר לכם לשנות את קובצי התבנית, את גליונות הסגנונות ואת קובצי Javascript של הערכה שאתם משתמשים בה כעת.

רשימת הקבצים מימין מחולקת לחמישה חלקים:

  • קובצי תבנית
  • קובצי CSS (גליונות סגנונות)
  • קובצי Javascript
  • קובצי Locales (קבצים המשמשים לתרגום)
  • קבצי PHP (יש לשנות בזהירות)

כל מקטע של קבצים עשויים לכלול ארבעה תת-מקטעים:

  • קבצים מותאמים אישית
  • קבצים השייכים לערכת נושא
  • קבצים השייכים לערכת האב
  • קבצים השייכים לתבנית התבנית

תיחום קובץ

כדי לשנות קובץ, לחצו על שמו. התוכן שלו יוצג בעורך. אם ניתן לעקוף את הקובץ, תוכלו לשמור אותו באמצעות לחיצה על הלחצן שמירה (ים).

שימו לב: שינויים חלים מיד לאחר השמירה.

אם תשנו קובץ שמגיע מתוך ערכת התבניות או ערכת האב, הקובץ יועתק לתוך הערכה הנוכחית שלכם.

הבלטת תחביר

ניתן להפעיל את הדגשת התחביר בהעדפות שלך (בלשונית האפשרויות שלי).

קריאה נוספת

כדי לערוך את קובצי התבנית, בדקו את רשימת של תגי התבנית.
