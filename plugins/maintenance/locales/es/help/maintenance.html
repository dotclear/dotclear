Ajustes de alerta

Activación

Marque esta casilla para mostrar mensajes de alerta en las tareas que lleven retraso o que nunca se hayan ejecutado.

Note: in Preferences, additional options allow you to display these tasks on your dashboard.

Frecuencia

Programa la visualización de alertas de tareas pendientes.

Esta frecuencia puede ser común a todas las tareas (seleccione el botón de opción Usar una hora de llamada para todas las tareas ) o puede establecerse para cada tarea.
