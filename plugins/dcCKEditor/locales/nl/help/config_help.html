Activering

Vink dit vakje aan om gerelateerde berichten op deze blog te gaan gebruiken.

Opmerking: als je de plugin deactiveert, worden eerdere keuzes in het beheergedeelte niet verwijderd.

Opties

Je kunt kiezen welke knoppen worden weergegeven. Let op, deze instellingen gelden voor alle gebruikers op het blog.

Hoe te gebruiken

Elke gebruiker op het blog moet de plugin kiezen in zijn voorkeuren.
