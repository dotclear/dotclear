Esta página enumera todos los usuarios del sistema.

Para añadir (o modificar) los permisos de los usuarios, se requerirá su contraseña en el campo Su contraseña.

Filtro de la lista de usuarios

Búsqueda
La búsqueda se basa en el ID de usuario. Se puede utilizar el comodín * para buscar sólo una parte del ID de usuario.
Ordenar por
La lista de usuarios puede ordenarse por su ID, su Apellido, su Nombre, su Nombre para mostrar (apodo) o el Número de entradas que han creado.
Clasificar
Indica el método de ordenación que se aplicará a la lista de resultados (Ascendente o Descendente).
Usuarios por página
Establece el número de usuarios mostrados en cada página de la lista de resultados.

Acciones por lotes de usuarios

Es posible realizar un lote de acciones sobre varios usuarios a la vez.

  • Establecer permisos: Establecer permisos para uno o más blog(s).
  • Borrar: Elimina del sistema al usuario o usuarios seleccionados. Un usuario que ha creado entradas no se puede eliminar.

Si decides establecer los permisos de los usuarios, serás redirigido a una nueva página con una lista de los blogs disponibles en tu sistema. Marque los blogs en los que desea establecer los permisos y pulse Establecer permisos.

Permisos de los usuarios

Para cada blog, puede establecer los siguientes permisos:

  • Administrador: Puede administrar el blog por completo.
  • Gestionar sus propias entradas y comentarios: El usuario puede crear entradas y gestionar los comentarios de las entradas que haya creado.
  • Publicar entradas y comentarios: El usuario puede modificar el estado de publicación de sus propias entradas y de los comentarios de sus entradas.
  • Borrar entradas y comentarios: El usuario puede borrar sus propias entradas así como los comentarios de sus entradas.
  • Gestionar todas las entradas y comentarios: El usuario puede gestionar todas las entradas de este blog, sin importar si es el autor o no.
  • Gestionar categorías: El usuario puede crear, modificar y eliminar categorías.
  • Gestionar sus propios elementos multimedia: El usuario puede añadir elementos multimedia y modificar o eliminar sus propios medios.
  • Gestionar todos los elementos multimedia: El usuario puede gestionar los elementos multimedia desde cualquier usuario.
  • Gestionar páginas: El usuario puede añadir y modificar páginas.
  • Gestionar el blogroll: El usuario puede gestionar el blogroll del blog actual.

El estado Superadministrador se establece en el perfil de usuario.
