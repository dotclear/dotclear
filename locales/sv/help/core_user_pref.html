Mina alternativ

Gränsgränssnitt
Aktivera förbättrad uppladdning i mediehanteraren
Gör det möjligt att aktivera det avancerade gränssnittet i mediehanteraren som gör det möjligt att ladda upp flera filer åt gången (Obs: Din webbläsare måste stödja Javascript för att kunna använda den här funktionen).
Inaktivera Javascript powered drag and drop för att beställa objekt
Tillåter att inaktivera "dra och släpp" för artikelsortering. Dra och släpp kommer att ersättas av numeriska fält som låter dig ange fältposition istället.
Antal element som visas per sida i mediehanteraren
Definierar antalet medier som kommer att visas i mediehanteraren (mappar beaktas inte när objekt räknas).
Använd inte standard favicon
Tillåter att inaktivera standard favicon i webbläsarens adressfält.
Upplaga
Föredraget format
Välj postens syntax. Wiki är en förenklad syntax och kommer att konverteras till giltig HTML. Om du inte har en perfekt förståelse för HTML, rekommenderar vi dig att välja Wiki syntax. Se syntaxreferensen för Wiki för mer information.
Standard poststatus
Standardstatus för nya poster kan ställas in till:
  • Väntande: Publiceringsstatus har ännu inte beslutats.
  • Schemalagd: Posten kommer att ställas in online vid den tidpunkt och datum som anges i Publicerad på fältet.
  • Avpublicerat: Offline post.
  • Publicerad: Online-post.
Ändra fältets höjd
Sätter höjden på versionsfältet för en post. Som standard är denna parameter satt till 24.
Aktivera WYSIWYG-läge
Tillåter användning av den visuella redigeraren (WYSIWYG: Vad du ser är vad du får) för poster, sidor och kategorier beskrivning fält.
Andra alternativ

Syntaxmarkering i tema-editorn

Min instrumentpanel

Mina favoriter
Denna zon visar de valda favoriterna på bloggens instrumentpanel. De kan beställas om med musen om Javascript är aktiverat i din webbläsare. Annars kan ett fält ge dem ett beställningsnummer. När du är klar, glöm inte att spara dina ändringar genom att trycka på Spara ordning.
Du kan också ta bort favoriter genom att välja dem och trycka på Ta bort valda favoriter.
Definiera som standardfavoriter (superadministratör)
Aktivera för att ställa in just nu definierade favoriter som standardlista för alla användare. Denna lista kommer att föreslås till användare om de inte har definierat sina egna favoriter ännu.
Andra tillgängliga favoriter
Alla plugins som gör det möjligt att sätta som favoriter delas här. Markera motsvarande rutor och lägg till dem i instrumentpanelen genom att trycka på Lägg till mina favoriter.
Visa favoriter högst upp på menyn
Om aktiverad, visar favoriter högst upp på menyn.
Dashboard moduler
Val som görs i denna zon kommer att återspeglas på din bloggs instrumentpanel.
Visa dokumentationslänkar
Aktivera för att visa dokumentationslänkar på instrumentpanelens landningssida.
Visa Dotclear nyheter
Aktivera för att visa Dotclear nyheter på instrumentpanelens landningssida.
Visa snabbformulär
Aktivera för att visa formuläret "quick entry" på instrumentbrädans landningssida.
Visa förfallna uppgifter räknare på underhåll instrumentpanelen ikonen
Möjliggör för att visa antalet försenade uppgifter (dvs att de ska behandlas) under Underhåll -ikonen. Underhållsikonen måste ställas in i dina favoriter för att använda denna funktion.
Visa listan över förfallna uppgifter på instrumentpanelens objekt
Aktivera för att visa listan över förfallna uppgifter (dvs som ska behandlas) i en dedikerad avdelning på Dashboard landningssidan.
