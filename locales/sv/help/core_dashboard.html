Rubrik

Om det finns flera bloggar kan du komma åt på denna instans, en rullgardinslista låter dig välja den du vill arbeta med. Om det bara finns en blogg, det helt enkelt visar sitt namn.

Följande länk tillåter att gå på den aktuella bloggen (öppnas i en ny flik).

De andra länkarna gör att du kan gå tillbaka till den här instrumentpanelen, komma åt dina inställningar och logga ut.

Anpassning av instrumentpanelen

Favoriter
De stora ikonerna i mitten av sidan kan du snabbt komma åt dina favorit admin sidor.
Dashboard moduler
Modulerna som visas under favoriterna installeras som standard eller läggs till av plugins.

You can personalize the display of favorites and modules by going in the My dashboard tab in the Preferences page.

Huvudmenyn

Du kan visa eller dölja denna meny när som helst genom att klicka på dess högra sida (eller genom att aktivera Göm huvudmenyn länk).

The first part of the menu lists all the favorite pages of the user. You can select to display this list or not in the My dashboard tab in the Preferences page.

Resten av menyn listar administrationssidorna. Den är uppdelad i tre delar:

  • Blogg för allt i förhållande till nuvarande blogginnehåll och parametrar.
  • Systeminställningar för att hantera den globala installationen.
  • Plugins för tillägg som påverkar hela systemet.
