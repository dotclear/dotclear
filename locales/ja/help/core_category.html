新しいカテゴリを作成

名前
新しいカテゴリ名を選択してください。このフィールドは必須です。
親
サブカテゴリを作成することができます。その場合は、 Parent フィールドでその親カテゴリを選択する必要があります。
URL
ドットクリアでは、カテゴリーのデフォルトURLを作成し、 http://myblog.tld/category/My-categoryのようなパスでアクセスできるようにしています。右側の小さなロックをクリックすると、好きなように変更することができます。
説明
この説明項目の内容は、ブログにカテゴリを入力するときに表示されます。 任意の種類のコンテンツ (段落、リストなど) で入力できます。このフィールドで使用されるフォーマットは HTML です。

このカテゴリを移動

親カテゴリ
サブカテゴリを作成することができます。その場合は、 Parent フィールドでその親カテゴリを選択する必要があります。
兄弟カテゴリ
カテゴリを親になる他のどのカテゴリにも移動できます。 その位置を他の兄弟に指定するには、その前または後に移動します。
