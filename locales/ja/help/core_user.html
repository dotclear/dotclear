ユーザープロフィール

ユーザー ID
アルファベット、数字または記号を使用する少なくとも2文字（スペースなし）。このフィールドは必須です。
新しいパスワード
パスワードを変更したい場合 New password と Confirm password フィールドに新しいパスワードを 2 回入力します。 パスワードは6文字以上でなければなりません。両方のフィールドは必須です。
接続にはパスワードの変更が必要です
新しいユーザーが接続するときに自分のパスワードを選択する必要があるように、このチェックボックスをオンにします。
Super administrator
これにより、現在のユーザーにシステム上でホストされているすべてのブログのすべての権限を与えます。
姓、名
表示名が指定されていない場合、表示される著者の名前には、これら2つのフィールドに入力された姓と名が含まれます。
表示名
記事に署名したいニックネームを選択できます。 空でない場合、このフィールドは姓と名のフィールドの代わりに使用されます。
Eメールアドレス
このメールアドレスは、Dotclearの認証画面からパスワードを回復する必要がある場合に使用されます。
URL
ユーザーのウェブサイトを示します。このフィールドが空でない場合、著者の名前はこのウェブサイトへのクリック可能なリンクになります。

オプション

インターフェイス
言語
新しいエントリのデフォルト言語を設定します。 この言語で翻訳が利用可能な場合は、ブログインターフェースもこの言語で翻訳されます。
Timezone
このオプションは、エントリの公開時間の表示方法に影響します。
版
優先する形式
エントリの構文を選択します。Wiki は簡略化された構文であり、有効な HTML に変換されます。HTML を完全に理解していない場合は、Wiki 構文を選択することをお勧めします。詳細については、Wiki 構文リファレンスを参照してください。
デフォルトのエントリステータス
新しいエントリの既定の状態を次に設定できます:
  • 保留中: 出版状況はまだ決定されていない。
  • スケジュール日: エントリは、 公開日時 フィールドで指定された日時にオンラインで設定されます。
  • 未発表: オフラインエントリ。
  • 公開日: オンラインエントリ。
Entry edit field height
項目のエディションフィールドの高さを設定します。デフォルトでは、このパラメータは 24 に設定されています。
タグリストの形式
項目を編集中に表示するかどうかを選択できます。 すべてのタグ（Extended）または最も使用されているもののみ（Short）。
