Referência de sintaxe Wiki

A sintaxe Wiki é uma maneira de melhorar seu texto com um conjunto mínimo de tags estudadas para cobrir as necessidades básicas (títulos, parágrafos, aspas, listas. .).

Elementos de bloco
  • Deixe uma linha vazia entre dois blocos similares.
  • Parágrafo: Texto gratuito, terminado por uma linha vazia se outro parágrafo for seguido.
  • Título: !!! título, !! título ou ! Título, permitindo que você use três níveis diferentes de cabeçalho .
  • Linha horizontal: ----
  • Lista: Comece cada linha com * ou # para listas numeradas ou numeradas, respectivamente. A imagem da lista é feita misturando os marcadores desta forma: 
        * item 1
    ** item 1.1
    * item 2
    *# item 2.1
...
  • Listas de definição: Comece cada linha com = para o termo a definir e com : para a definição do termo. 
      = termo
  : descrição do termo
  • Texto pré-formatado: Cada linha precisa começar com um espaço.
  • Taxa de bloco: Cada linha deve começar com um >.
  • Bloco lateral: Cada linha deve começar com um ).
Tags de formatação
  • Emphasis: Duas citações ''texto''
  • Forte ênfase: Dois sublinhados __text__
  • Realçado: Duas aspas duplas ""texto""
  • Nova linha: %%%
  • Inserção: Dois mais +text++
  • Deleção: Dois minutos --text--
  • Link: [url], [name³url], [name├url£language] ou [name├url³language]
  • Imagem: ((url texto alternativo)), ((url├texto alternativo)) ou ((url➲ alternativa de posicionamento/longa)).
    A posição pode ser L (esquerda), R (direita) ou C (centrado).
  • Âncora: ~anchor~
  • Acronym: ??acronym|title??
  • HTML embutido: Duas aspas anteriores ``HTML code``
  • Citação embutida: {{quote}}, {{quote|language}} ou {{quote|language|url}}
  • Código: @@code@@
  • Notas de rodapés: $$footnote$$
  • Sobrescrito : ^superscript^
  • Subscript : ,,,subscript,,
Texto não formatado
Se você não quer que um caractere de formatação seja interpretado como tal, adicione um \ pouco antes dele. Por exemplo: \[Texto entre colchetes sem ser um link\]
Código HTML
Mesmo se você escolheu a sintaxe Wiki, algumas vezes você pode precisar de uma formatação mais poderosa, ou seja, sintaxe HTML. Faça-o da seguinte maneira: 
///html
<p style="color:red">meu texto em vermelho</p>
///
