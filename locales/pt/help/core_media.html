Lista de media

A página principal do gestor de multimédia exibe a lista de ficheiros disponíveis (os medias) para o blog actualmente seleccionado.

Basta clicar nos ícones ou nomes do media na pasta actual ou qualquer sub-pasta para exibi-los.

Seleccionar um ficheiro MP3 abra imediatamente o leitor de multimédia. O plugin Flash deve estar instalado em seu navegador para que o leitor funcione.

As imagens são exibidas com uma miniatura quando possível.

Adicionar ficheiros

Escolha o ficheiro
Here you can select one of your local files to send it on your blog. Should the file be bigger than the maximum size, it will not be sent.
Título
Título opcional do ficheiro que deseja enviar.
Privado
Check this if you want the file to be private, i.e. accessible only by its owner and not accessible from the blog (until published).
Envio aprimorado
Allows to select one or several of your local files to send them on your blog. Should any file be bigger than the maximum size, it will not be sent.
When you select several local files, they are added to the queue. You can add more files by clicking again on Choose files. You can also remove files from the queue by clicking on the corresponding red cross-shaped icon.
When you are ready, click on Upload.

Nova pasta

Este formulário permite criar uma nova pasta dentro da pasta actual. Basta escolher um nome para ele, como você poderia fazer em seu próprio computador.

Detalhes do conteúdo multimédia

This form allows you to create a new subfolder in the currently displayed folder. Just choose a name for it as you would do on your own computer.

Tratamento específico para imagens

Quando você adiciona uma imagem, dependendo do seu tamanho, até quatro versões diferentes serão disponibilizadas:

Nome do ficheiro
Alterar o nome do arquivo do arquivo atual.
Título do ficheiro
Dê um título ao arquivo ou altere o atual.
Data do ficheiro
Alterar a data do arquivo.
Privado
Especifique que o arquivo será ocultado para qualquer pessoa, exceto para seu dono no gerenciador de mídia.
Nova pasta
Permite que você mova o arquivo para outra pasta
Modificar o ficheiro
With this form, you can replace the file while keeping all the attributes (name, title...). As always, you have to respect the maximum file size.

Adicionar ficheiros numa entrada

Pode facilmente anexar um ficheiro em suas entradas.
Primeiro, crie uma nova entrada e guardá-a.
Em seguida, clique em "Adicionar ficheiros nesta entrada" para abrir o gestor de multimédia.
Clicar no ícone [+](Anexar este ficheiro à entrada).
O ficheiro será, então, anexado a entrada.

  • quadrado: um quadrado de 48x48 pixeis
  • miniatura: o lado maior é reduzido para 100 pixeis.
  • pequeno: o lado maior é reduzido para 240 pixeis.
  • médio: o lado maior é reduzido para 500 pixeis.
  • original: a imagem mantém-se inalterado

Os anexos serão mostrados na página das entradas, como lista ao lado do conteúdo. Os ficheiros MP3 serão disponibilizados através do Flash player, para que eles possam ser facilmente lidos pelos visitantes.

Podcast e broadcast de ficheiros musicais

Todos os ficheiros anexados serão incluídos em seus feeds RSS, permitindo o fácil podcasting de qualquer tipo de ficheiros.

Just click on the [+] icon (Attach this file to entry). The file will then be attached to the post.

Os arquivos anexados serão exibidos na página do post como uma lista após o conteúdo do post. Os arquivos MP3 serão disponibilizados através de um Flash Player para que possam ser facilmente tocados pelos seus visitantes.

Podcast e transmissão de arquivos de música

All attached files will be included in your blog feeds, allowing an easy podcasting of any type of file.
