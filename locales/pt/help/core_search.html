Opções da pesquisa

Especifique as palavras ou termos que você está procurando no campo de texto e selecione o escopo de pesquisa:

Pesquisar em entradas
A pesquisa só será feita nos campos título do artigo, Resumo e Conteúdo de uma entrada.
Pesquisar nos comentários
Só será feita uma pesquisa nos comentários.

Resultados da pesquisa

Resultado

Um resultado de pesquisa permite que você clique em um do link do artigo para editá-lo.

Ações em lote

É possível fazer um lote de ações de vários artigos em um só vez. As ações possíveis dependem das permissões do usuário:

  • Publicar: Publique o artigo.
  • Despublicar: Coloque o artigo offline.
  • Agendar: Agende as postagens a serem colocadas online na data de publicação.
  • Marcar como pendente: Pendente de ser publicado.
  • Marcar como selecionado: Destacar as entradas selecionadas.
  • Marcar como não selecionado: Remover destaque (se houver) nas entradas selecionadas.
  • Alterar categoria: Redirecione na lista de categorias para alterar a categoria dos artigos selecionados.
  • Mude o idioma: Redirecione na lista de idiomas para modificar os idiomas de entrada.
  • Alterar o autor: Permite modificar o autor das postagens especificando o ID do usuário do novo autor.
  • Apagar: Exclua as entradas. Uma janela pop-up será exibida para confirmar sua escolha.
  • Adicionar tags: Adicionar tags às postagens selecionadas.
  • Remova as tags: Remova uma ou mais tags das postagens selecionadas.
