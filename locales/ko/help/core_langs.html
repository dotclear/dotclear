설치된 언어

여기에서 관리 프로필과 블로그에 사용할 수 있는 설치된 언어 목록을 확인할 수 있습니다. 이러한 각 언어는 제거할 수 있습니다.

언어가 없는 경우 블로그뿐만 아니라 관리 부분에서도 기본적으로 영어가 사용됩니다.

언어 설치 또는 업그레이드

사용 가능한 언어

여기에서 닷클리어 웹사이트에서 사용 가능한 언어 목록을 확인할 수 있습니다. 이러한 언어 중 하나를 설치하려면 드롭다운 목록에서 해당 언어를 선택하고 비밀번호를 입력한 후 언어 설치를 누르세요.

그러면 선택한 언어가 다운로드되어 시스템에 설치됩니다.

zip 파일 업로드

압축된 아카이브(zip 형식)를 사용하여 언어를 설치할 수도 있습니다. 컴퓨터에서 해당 파일을 선택하고 비밀번호를 입력한 후 언어 업로드를 누릅니다.
