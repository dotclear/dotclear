Criar uma nova categoria

Sobrenome
Escolha o nome que você deseja dar à sua nova categoria. Este campo é obrigatório.
Antecessor
É possível criar subcategorias. Nesse caso, você tem que selecionar sua categoria pai no campo Pai.
URL:
Dotclear cria uma URL padrão para a categoria, tornando-o acessível com um caminho como http://myblog. ld/categoria/meu-categoria. Ao clicar no pequeno bloqueio à direita, você pode modificá-lo da forma que quiser.
Descrição:
Este conteúdo do campo descrição será exibido quando inserir uma categoria no blog. Você pode preenchê-lo com qualquer tipo de conteúdo (parágrafos, listas, etc.). O formato utilizado para este campo é HTML.

Mover esta categoria

Categoria Parental
É possível criar subcategorias. Nesse caso, você tem que selecionar sua categoria pai no campo Pai.
Categoria de irmãos
Você pode mover a categoria para qualquer outra, que se tornará seu pai. Você também pode especificar sua posição para seus outros irmãos movendo-a antes ou depois de um deles.
