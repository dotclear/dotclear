Plugin yang terinstal

Setiap baris dimulai dengan logo plugin yang akan Anda temukan di menu (di sebelah kiri layar Anda).

Logo Dotclear di sebelah tombol aksi menunjukkan bahwa plugin ini hadir langsung dengan Dotclear dan merupakan salah satu komponennya.

Plugin yang diaktifkan

Daftar ini menampilkan informasi tentang plugin yang diaktifkan: Nama, versi, deskripsi singkat. Ada dua tindakan yang dapat dilakukan: Menonaktifkan (operasi sementara) atau Menghapus (operasi permanen).

Plugin yang dinonaktifkan

Daftar ini menunjukkan plugin yang telah dinonaktifkan oleh pengguna. Untuk setiap plugin, ada dua tindakan yang dapat dilakukan: Mengaktifkan atau Menghapus (operasi permanen).

Menambahkan plugin

Halaman ini memungkinkan Anda untuk menelusuri repositori DotAddict dan menginstal plugin langsung dari sana.

Tersedia dua metode pencarian:

  • Dengan mengetikkan nama plugin di kolom pencarian di repositori ,
  • Dengan menelusuri indeks

Dalam indeks, membuka sebuah plugin dengan mengklik tanda panah di sebelah namanya akan memungkinkan Anda untuk mengakses halaman DotAddict-nya atau pergi ke situs web dukungan jika penulis plugin memilihnya.

Instal atau tingkatkan secara manual

Anda dapat menginstal plugin Anda sendiri atau plugin yang tidak tersedia di DotAddict menggunakan halaman ini.

Ada dua cara untuk memasangnya:

  • Unggah file zip dari komputer Anda. Klik Jelajahi untuk memilih file, masukkan kata sandi Anda, dan tekan Unggah.
  • Unduh file zip langsung dari Web dengan memasukkan URL-nya, lalu masukkan kata sandi Anda dan tekan Unduh.
