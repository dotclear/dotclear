Profil pengguna

ID Pengguna
Minimal 2 karakter menggunakan huruf, angka, atau simbol (tanpa spasi). Kolom ini wajib diisi.
Kata sandi baru
Jika Anda ingin mengubah kata sandi, masukkan kata sandi baru dua kali di Kata sandi baru dan Konfirmasi kata sandi kolom. Kata sandi harus terdiri dari setidaknya 6 karakter. Kedua kolom tersebut wajib diisi.
Diperlukan perubahan kata sandi untuk menyambung
Centang kotak ini agar pengguna baru harus memilih kata sandinya sendiri saat akan terhubung.
Administrator super
Ini akan memberikan pengguna saat ini semua izin pada semua blog yang dihosting di sistem.
Nama belakang, Nama depan
Jika tidak ada nama tampilan yang disediakan, nama penulis yang ditampilkan akan berisi nama belakang dan nama depan yang dimasukkan dalam dua bidang ini.
Nama tampilan
Anda dapat memilih nama panggilan yang ingin Anda gunakan untuk menandatangani artikel Anda. Jika tidak kosong, kolom ini akan digunakan sebagai pengganti kolom nama belakang dan nama depan.
Email
Alamat email ini akan digunakan jika Anda perlu memulihkan kata sandi Anda dari layar autentikasi Dotclear.
URL
Menunjukkan situs web pengguna. Jika bidang ini tidak kosong, nama penulis akan menjadi tautan yang dapat diklik ke situs web ini.

Pilihan

Antarmuka
Bahasa
Mengatur bahasa default untuk entri baru. Jika terjemahan tersedia untuk bahasa ini, antarmuka blog akan diterjemahkan dalam bahasa ini juga.
Zona waktu
Opsi ini berdampak pada cara waktu publikasi entri ditampilkan.
Edisi
Format yang disukai
Choose the entry's syntax. Wiki is a simplified syntax and will be converted to valid HTML. Unless you have a perfect understanding of HTML, we advise you to choose the Wiki syntax. See the Wiki syntax reference for more information.
Status entri default
Status default untuk entri baru dapat diatur ke:
  • Tertunda: Status publikasi belum diputuskan.
  • Terjadwal: Entri akan ditetapkan secara online pada waktu dan tanggal yang disediakan di Dipublikasikan di bidang .
  • Tidak dipublikasikan: Entri offline.
  • Diterbitkan: Entri online.
Tinggi bidang edit entri
Mengatur tinggi bidang edisi entri. Secara default, parameter ini diatur ke 24.
Format daftar tag
Anda dapat memilih apakah Anda ingin melihat, ketika mengedit entri, semua tag yang digunakan (Extended) atau hanya tag yang paling sering digunakan (Short).
