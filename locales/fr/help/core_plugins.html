Plugins installés

Chaque ligne commence par le logo du plugin que vous trouverez dans le menu (à gauche de votre écran).

La présence d'un logo Dotclear à côté des boutons d'action indique que le plugin est fourni directement avec Dotclear et qu'il fait partie intégrante de celui-ci.

Plugins activés

Cette liste affiche des informations sur les plugins activés : nom, version, brève description. Deux actions sont possibles : Désactiver (opération temporaire) ou Supprimer (opération définitive).

Plugins désactivés

Cette liste répertorie les plugins qui ont été désactivés par l'utilisateur. Pour chacun d'entre eux, deux actions sont possibles : Activer ou Supprimer (opération définitive).

Ajouter des plugins

Cette page vous permet de naviguer dans le dépôt DotAddict et d'installer un plugin directement à partir de celui-ci.

Deux méthodes de recherche sont disponibles :

  • En saisissant le nom du plugin dans le champ « Rechercher dans le référentiel »,
  • En parcourant l'index

Dans l'index, en développant un plugin en cliquant sur la flèche située à côté de son nom, vous pourrez accéder à sa page DotAddict ou vous rendre sur un site d'assistance si l'auteur du plugin en a choisi un.

Installer ou mettre à jour manuellement

Cette page vous permet d'installer vos propres plugins ou des plugins qui ne sont pas disponibles sur DotAddict.

Il existe deux façons de les installer :

  • Téléchargez un fichier ZIP depuis votre ordinateur. Cliquez sur « Parcourir » pour sélectionner le fichier, saisissez votre mot de passe, puis cliquez sur « Télécharger ».
  • Téléchargez un fichier ZIP directement depuis Internet en saisissant son URL, puis entrez votre mot de passe et cliquez sur « Télécharger ».
