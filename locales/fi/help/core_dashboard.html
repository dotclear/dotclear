Otsikko

Jos tässä instanssissa on useita blogeja, voit valita pudotusvalikosta sen, jota haluat käsitellä. Jos blogeja on vain yksi, näytetään vain sen nimi.

Seuraavan linkin kautta pääset nykyiseen blogiin (avautuu uuteen välilehteen).

Muiden linkkien avulla voit palata kojelaudalle, käyttää asetuksiasi ja kirjautua ulos.

Kojelaudan personointi

Suosikit
Sivun keskellä olevien suurten kuvakkeiden avulla pääset nopeasti suosikkisivuillesi.
Ohjaustaulun moduulit
Suosikkien alla näkyvät moduulit on asennettu oletusarvoisesti tai lisätty lisäosilla.

You can personalize the display of favorites and modules by going in the My dashboard tab in the Preferences page.

Päävalikko

Voit näyttää tai piilottaa tämän valikon milloin tahansa napsauttamalla sen oikeaa puolta (tai aktivoimalla Piilota päävalikko -linkin).

The first part of the menu lists all the favorite pages of the user. You can select to display this list or not in the My dashboard tab in the Preferences page.

Valikon loppuosassa luetellaan hallintasivut. Se on jaettu kolmeen osaan:

  • Blogi kaikelle, mikä liittyy nykyiseen blogin sisältöön ja parametreihin.
  • Järjestelmäasetukset maailmanlaajuisen asennuksen hallintaan.
  • Liitännäiset koko järjestelmään vaikuttavia laajennuksia varten.
