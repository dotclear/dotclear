Lista media

The media manager's main page displays a list of available files (medias) for the currently selected blog.

You can click the icons or names of the medias in the folder or any subfolder to display them.

Selecting a MP3 sound file displays a player to immediately play the file. The Flash plugin must be installed on your browser for this player to operate.

Le immagini vengono visualizzate con una miniatura, quando possibile.

Aggiungi file

Scegli un file
Qui puoi selezionare uno dei tuoi file locali per inviarlo al tuo blog. Qualora il file superi la dimensione massima non verrà inviato.
Titolo
Titolo opzionale per il file da inviare.
Privato
Seleziona questa opzione se vuoi che il file sia privato, cioè accessibile solo dal suo proprietario e non accessibile dal blog (fino alla pubblicazione).
File name
Permette di selezionare uno o più file locali per inviarli al tuo blog. Se un file supera la dimensione massima, non verrà inviato.
Quando selezioni più file locali, questi vengono aggiunti alla coda. Puoi aggiungere altri file facendo nuovamente clic su Scegli file. Puoi anche rimuovere i file dalla coda facendo clic sulla corrispondente icona a forma di croce rossa.
Quando sei pronto, fai clic su Carica.

Elimina file

This form allows you to create an new subfolder in the currently displayed folder. Just choose a name for it as you would do on your own computer.

Nuova cartella

Questo modulo consente di creare una nuova sottocartella nella cartella attualmente visualizzata. Scegli semplicemente un nome come faresti sul tuo computer.

Dettagli media

When you add an image, depending of its size, up to four versions will be made available:

Nome file
Cambia il nome del file corrente.
Titolo del file
Dai un titolo al file o cambia quello attuale.
Data del file
Cambia la data del file.
Privato
Specificare che il file sarà nascosto a chiunque tranne il suo proprietario nel media manager.
Nuova directory
Consente di spostare il file in un'altra cartella.
Cambia file
Con questo modulo è possibile sostituire il file mantenendo tutti gli attributi (nome, titolo...). Come sempre, devi rispettare la dimensione massima del file.

Caso specifico: immagini

You can easily attach a file to one of your posts. First, create a new post and save it. Then click on "Add files to this entry" to open the media manager.

  • square: a 48x48 pixels square thumbnail,
  • thumbnail: the image reduced to 100 pixels on its longest side.
  • small: the longuest side is reduced to 240 pixels.
  • medium: the longuest side is reduced to 500 pixels.
  • original: the original, unchanged image.

Just click on the [+] icon (Attach this file to entry). The file will then be attached to the post.

Allega un file a una voce

The attached files will be shown on the post page, as a list beside the post content. The MP3 files will be made available through the Flash player, so that they can easily by read by visitors.

Basta fare clic sull'icona [+] (Allega questo file alla voce). Il file verrà quindi allegato al post.

I file allegati verranno visualizzati nella pagina post, come un elenco dopo il contenuto del post. I file MP3 saranno resi disponibili attraverso un lettore Flash, in modo che possano essere facilmente riprodotti dai vostri visitatori.

Podcast e trasmissione di file musicali

Tutti i file allegati verranno inclusi nei feed del tuo blog, consentendo un facile podcasting di qualsiasi tipo di file.
