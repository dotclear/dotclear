Aggiornamento dei temi

Quando le nuove versioni dei temi installati sul tuo blog sono state caricate nel centro risorse Dotaddict, the page has an additional tab named Theme Updates, che elenca i temi con il numero di versione installato e il numero di versione dell'aggiornamento.

Quindi basta usare il pulsante azione Aggiorna.

Se sono disponibili diversi aggiornamenti, un pulsante di azione in fondo alla pagina permette di Aggiornare tutti i temi contemporaneamente.

Si prega di notare: eventuali modifiche apportate ai file del tema andranno perse durante l'aggiornamento del tema.

Temi installati

In questa scheda viene visualizzato l'elenco dei temi installati. Puoi selezionare un tema cliccando su di esso e premendo Usa questo.

Il tema attualmente selezionato può essere modificato usando il pulsante Modifica file tema . Se il tema selezionato include un editor di temi, può essere facilmente personalizzato utilizzando il pulsante Configura tema.

Puoi anche disattivare un tema (per esempio se hai problemi di compatibilità con esso). Apparirà una scheda temi disattivati contenente la lista di questi.

Aggiungi temi

Questa scheda ti permette di sfogliare il repository dei temi DotAddict sia per ordine alfabetico, sia per ricerca usando il campo dedicato. I temi già installati non appaiono nella lista.

Installa o aggiorna manualmente

Se il tema che si desidera installare (o aggiornare) non è nel repository DotAddict, è possibile installarlo sia caricandolo dal disco rigido, sia inserendo il suo URL di installazione. Un tema è sempre confezionato come file zip.
