Il mio profilo

Name, First name
Se il nome visualizzato è vuoto, il nome dell'autore verrà visualizzato come nome e cognome come impostato qui.
Nome visualizzato
Qui puoi scegliere un nickname che verrà visualizzato nei tuoi post al posto del tuo cognome e nome.
Email
Indirizzo e-mail utilizzato quando richiedi una nuova password.
Anche le notifiche di nuovi commenti verranno inviate a questo indirizzo.(Se aggiorni questo campo, devi inserire la tua password attuale nell'apposito campo in fondo al modulo)
URL
Il sito web dell'utente. Se impostato, il nome dell'autore verrà visualizzato come collegamento al suddetto sito web.
Preferred format
La lingua utilizzata durante la pubblicazione delle voci. Se la traduzione è disponibile, l'interfaccia verrà visualizzata anche in quella lingua.
Default entry status
Tale scelta definirà il tempo visualizzato sulla pubblicazione postale.
Entry edit field height
Per modificare la password, annota due volte la tua password nei campi Nuova password e Conferma password. La lunghezza minima della password è di 6 caratteri. Se non viene fornita una nuova password, viene mantenuta quella attuale. (Se aggiorni questo campo, devi inserire la tua password attuale nell'apposito campo in fondo al modulo)

Le mie opzioni

Interfaccia
Attiva uploader migliorato nel gestore multimediale
Consente di attivare l'interfaccia avanzata nel gestore multimediale che consente di caricare più file contemporaneamente (Nota: il browser Web deve supportare Javascript per poter utilizzare questa funzionalità).
Disabilita il trascinamento e il rilascio di Javascript per ordinare oggetti
Consente di disabilitare il "trascinamento della selezione" per l'ordinamento degli articoli. Il trascinamento della selezione verrà sostituito da campi numerici che ti consentiranno invece di inserire la posizione dei campi.
Numero di elementi visualizzati per pagina nel gestore multimediale
Definisce il numero di media che verranno visualizzati nel gestore media (le cartelle non vengono prese in considerazione durante il conteggio degli elementi).
Non utilizzare favicon standard
Consente di disabilitare la favicon predefinita nella barra degli indirizzi del browser Web.
Edizione
Formato preferito
Scegli la sintassi della voce. Wiki ha una sintassi semplificata e verrà convertito in HTML valido. A meno che tu non abbia una perfetta conoscenza dell'HTML, ti consigliamo di scegliere la sintassi Wiki. Per ulteriori informazioni, vedere il riferimento alla sintassi del Wiki.
Stato della voce predefinita
Lo stato predefinito per le nuove voci può essere impostato su:
  • In attesa: Lo stato di pubblicazione non è stato ancora deciso.
  • Scheduled: The entry will be set online at the time and date provided in the Published on field.
  • Non pubblicato: Voce offline.
  • Pubblicato: Voce online.
Altezza campo modifica immissione
Imposta l'altezza del campo di edizione di una voce. Per impostazione predefinita, questo parametro è impostato a 24.
Abilita la modalità WYSIWYG
Consente l'uso dell'editor visivo (WYSIWYG: Quello che vedi è quello che ottieni) per le voci, le pagine e le categorie dei campi di descrizione.
Altre opzioni
Formato elenco etichette
Puoi selezionare se vuoi vedere, mentre modifichi una voce, tutti i tag utilizzati (Extended) o solo quelli più utilizzati (Short).
Evidenziazione della sintassi nell'editor del tema
Abilita l'evidenziazione della sintassi quando si utilizza l'editor di temi (file HTML e CSS)

La mia dashboard

I miei preferiti
Questa zona mostra i preferiti scelti sulla dashboard del blog. Puoi riordinarli con il mouse se Javascript è attivato nel tuo browser. Altrimenti, un campo permette di attribuire loro un numero d'ordine. Una volta terminato, non dimenticare di salvare le modifiche premendo Salva ordine.
Puoi anche eliminare i preferiti selezionandoli e premendo Elimina i preferiti selezionati.
Definisci come preferiti predefiniti (super amministratore)
Abilita per impostare l'elenco dei preferiti attualmente definito come lista dei preferiti predefiniti per tutti gli utenti. Questo elenco sarà proposto agli utenti se non hanno ancora definito i propri preferiti.
Altri preferiti disponibili
Tutti i plugin che permettono di essere messi come preferiti sono giocati qui. Controllare le caselle corrispondenti e aggiungerle alla dashboard premendo Aggiungi tp i miei preferiti.
Mostra i preferiti in cima al menu
Se abilitata, visualizza i preferiti nella parte superiore del menu.
Moduli pannello impostazioni
Le scelte fatte in questa zona saranno riflesse sulla dashboard del tuo blog.
Visualizza link alla documentazione
Abilita per visualizzare i link di documentazione nella pagina di atterraggio della dashboard.
Mostra notizie Dotclear
Abilita per visualizzare le notizie Dotclear sulla pagina di atterraggio della dashboard.
Visualizza il modulo di inserimento rapido
Abilita per visualizzare il modulo "voce rapida" nella pagina di atterraggio della dashboard.
Visualizza il contatore delle attività scadute sull'icona del cruscotto di manutenzione
Consente di visualizzare il numero di attività scadute (cioè che dovrebbero essere elaborate) nell'icona Manutenzione . L'icona Manutenzione deve essere impostata nei tuoi preferiti per utilizzare questa funzione.
Visualizza l'elenco delle attività scadute sugli elementi della dashboard
Abilita per visualizzare la lista delle attività scadute (cioè che dovrebbero essere elaborate) in una sezione dedicata nella pagina di destinazione della Dashboard.
