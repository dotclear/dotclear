สำรวจอินเทอร์เฟซ

ความช่วยเหลือตามบริบท

นอกเหนือจากหน้าช่วยเหลือทั่วโลกแล้ว หน้าการจัดการหลายหน้ายังมีหน้าช่วยเหลือออนไลน์ให้ด้วย หาก JavaScript เปิดใช้งานอยู่ในเบราว์เซอร์ของคุณ คุณสามารถดูหน้าช่วยเหลือออนไลน์ได้ที่มุมขวาบนของหน้าเว็บโดยคลิกที่ลิงก์ ต้องการความช่วยเหลือหรือไม่?หาก JavaScript ถูกปิดใช้งาน หน้าช่วยเหลือจะแสดงอยู่ที่ด้านล่างของหน้าเว็บ

ในทุกหน้าการจัดการ คุณจะพบแถบด้านบน เมนูหลัก และส่วนท้ายเดียวกัน(ตำแหน่งของแต่ละองค์ประกอบเหล่านี้แสดงไว้ที่นี่สำหรับการดูบนคอมพิวเตอร์; บนอุปกรณ์มือถือหรือแท็บเล็ต ตำแหน่งจะแตกต่างกันไปขึ้นอยู่กับพื้นที่ที่มีอยู่)

แบนเนอร์ด้านบน

ส่วนที่อยู่ด้านบนของแต่ละหน้าประกอบด้วยสามองค์ประกอบ:

โลโก้

โดยค่าเริ่มต้น โลโก้นี้คือโลโก้ของ Dotclear และเชื่อมโยงไปยังหน้าแรกสำหรับการจัดการ อย่างไรก็ตาม ผู้ดูแลระบบแพลตฟอร์มสามารถเลือกโลโก้ที่แตกต่างและเปลี่ยนเส้นทางไปยังหน้าอื่น (โดยปกติจะเป็นหน้าแรกของบริการ)

บล็อก

องค์ประกอบแรกหลังจากโลโก้จะแสดงบล็อกที่กำลังใช้งานอยู่ซึ่งก็คือบล็อกที่คุณกำลังทำงานอยู่ในขณะนี้ หากการติดตั้งของคุณประกอบด้วยบล็อกหลายบล็อก ข้อมูลนี้จะแสดงในเมนูแบบเลื่อนลง ซึ่งช่วยให้คุณสามารถสลับไปยังบล็อกอื่นได้

ลิงก์ที่อยู่ถัดจากชื่อบล็อกจะช่วยให้คุณดูบล็อกในหน้าต่างหรือแท็บใหม่

ผู้ใช้

ที่ด้านขวาสุดของแผงด้านบน ลิงก์ "แดชบอร์ดของฉัน" จะนำคุณกลับไปยังหน้าแรกของบล็อก

ลิงก์ถัดไป,ความชอบของฉัน, จะพาคุณไปยังหน้าที่คุณสามารถแก้ไขข้อมูลส่วนตัวของคุณ, ตั้งค่าแดชบอร์ดของคุณ, และเลือกอินเตอร์เฟซและตัวเลือกการแก้ไขของคุณ.

… น่าแปลกใจที่ลิงก์ "ออกจากระบบ [ชื่อผู้ใช้ของคุณ]" ช่วยให้คุณออกจากระบบได้ ใช่แล้ว จริงๆ นะ (ที่ Dotclear เราเก่งเรื่องการใช้งานตามหลักการยศาสตร์มาก)

เมนูหลัก

เมนูหลักตั้งอยู่ทางด้านซ้ายของหน้าจอ สามารถยุบได้โดยการคลิกที่ขอบด้านขวา (และขยายได้โดยการคลิกที่ขอบเดียวกัน)

มันนำไปสู่หน้าการจัดการต่างๆ ที่สอดคล้องกับสิทธิ์ของคุณบนบล็อกที่ใช้งานอยู่ ประกอบด้วยสี่ส่วน 

  • รายการโปรดจะแสดงหน้าที่คุณได้ทำเครื่องหมายไว้เป็นรายการโปรดใน แท็บแดชบอร์ดของฉันของตัวเลือกของฉัน หากคุณยังไม่ได้เลือกอะไรไว้ รายการนี้จะแสดงชุดค่าเริ่มต้นของ Dotclear หรือของผู้ดูแลระบบแพลตฟอร์ม หากคุณไม่ต้องการแสดงบล็อกนี้ ให้ยกเลิกการเลือกช่องที่สอดคล้องกันในตัวเลือกแดชบอร์ดของคุณ
  • บล็อกแสดงรายการหน้าทั้งหมดที่อนุญาตให้คุณอัปเดตและกำหนดค่าบล็อกปัจจุบัน ไม่ว่าจะเป็นการแก้ไขโพสต์ จัดการสื่อ เลือกหน้าตาของบล็อก หรือปรับการตั้งค่าอื่นๆ
    หากคุณไม่มีสิทธิ์ผู้ดูแลระบบสำหรับบล็อกปัจจุบัน จะไม่มีบล็อกอื่นๆ ปรากฏอยู่ด้านล่างนี้ในเมนูหลัก
  • การตั้งค่าระบบให้การเข้าถึงหน้าต่าง ๆ สำหรับการจัดการการติดตั้ง Dotclear: การจัดการบล็อก, ผู้ใช้และภาษาที่มีให้ใช้, การติดตั้งส่วนขยาย, และการตั้งค่าระบบ. หากคุณมีสิทธิ์สำหรับบล็อกหลายบล็อก แต่ไม่มีสถานะผู้ดูแลระบบระดับสูง, จะประกอบด้วยเพียงรายการบล็อกเท่านั้น.
  • ปลั๊กอินช่วยให้คุณสามารถกำหนดค่าการตั้งค่าสำหรับปลั๊กอินที่ติดตั้งไว้ได้ หากการตั้งค่าของปลั๊กอินมีผลต่อบล็อกเพียงบล็อกเดียวในแต่ละครั้ง หน้าการจัดการของปลั๊กอินนั้นจะสามารถเข้าถึงได้จากส่วนบล็อกอย่างไรก็ตาม หากการตั้งค่ามีผลต่อทุกหน้าในระบบการติดตั้ง ปลั๊กอินนั้นจะปรากฏอยู่ที่นี่ และสามารถเข้าถึงได้เฉพาะผู้ใช้ที่มีสิทธิ์ผู้ดูแลระบบระดับสูงเท่านั้น

ส่วนท้าย

เวอร์ชันฟรีของ Dotclear มีให้สำหรับผู้ใช้ทุกคนที่สามารถค้นพบความลับของส่วนท้ายได้

ความช่วยเหลือสำหรับผู้เริ่มต้นอย่างสมบูรณ์

หากคุณไม่คุ้นเคยกับบล็อก เราขอแนะนำให้ศึกษาเอกสารประกอบของเรา (สารบัญฉบับสมบูรณ์สามารถดูได้ในแท็บเอกสารประกอบบนหน้านี้)

ฟอรัม Dotclearเป็นสถานที่สำหรับการสนับสนุนซึ่งกันและกันที่นี่ไม่มีคำถามที่โง่ อย่าลังเลที่จะเข้าร่วมหากคุณต้องการความช่วยเหลือ และทำไมไม่ช่วยผู้อื่นเป็นการตอบแทนด้วยล่ะ

เราพยายามใช้คำศัพท์ที่เข้าใจได้โดยไม่ต้องมีความรู้มาก่อน อย่างไรก็ตาม หากเราได้ใช้คำศัพท์ทางเทคนิคโดยไม่ตั้งใจ คุณอาจพบคำอธิบายได้ในอภิธานศัพท์ของเอกสารออนไลน์ของเรา

การเข้าถึงและการใช้งาน

อินเทอร์เฟซของ Dotclear ตรงตามเกณฑ์WCAG 2.0เกือบทั้งหมด รายงานโดยละเอียดจะเผยแพร่บนบล็อกของโครงการในเร็วๆ นี้ ในที่สุด เราต้องการที่จะปฏิบัติตามเกณฑ์ATAG 2.0อย่างสมบูรณ์ด้วย

การเข้าถึงและการใช้งานเป็นประเด็นสำคัญสำหรับทีม Dotclear เราทำการวิเคราะห์อย่างสม่ำเสมอด้วยความช่วยเหลือจากผู้เชี่ยวชาญ และดำเนินการแก้ไขและปรับปรุงอย่างต่อเนื่อง

คุณสามารถช่วยเราได้โดยการเปิดตั๋วที่ หรือเข้าร่วมในฟอรัม คุณยังสามารถเข้าร่วมการทดสอบผู้ใช้หรือเข้าร่วมทีมของเราได้อีกด้วย!
