โปรไฟล์ของฉัน

นามสกุล, ชื่อ
หากชื่อที่แสดงว่างเปล่า ชื่อของผู้แต่งจะแสดงเป็นชื่อแรกและนามสกุลตามที่ตั้งค่าไว้ที่นี่
ชื่อที่แสดง
ที่นี่คุณสามารถเลือกชื่อเล่นที่จะแสดงในโพสต์ของคุณแทนชื่อสกุลและชื่อแรกของคุณ
อีเมล
ที่อยู่อีเมลที่ใช้เมื่อคุณขอรหัสผ่านใหม่
การแจ้งเตือนความคิดเห็นใหม่จะถูกส่งไปยังที่อยู่นี้ด้วย(หากคุณอัปเดตข้อมูลนี้ คุณต้องป้อนรหัสผ่านปัจจุบันของคุณในช่องที่เหมาะสมที่ด้านล่างของแบบฟอร์ม)
URL
เว็บไซต์ของผู้ใช้ หากมีการตั้งค่า ชื่อผู้แต่งจะแสดงเป็นลิงก์ไปยังเว็บไซต์ดังกล่าว
ภาษาสำหรับอินเทอร์เฟซของฉัน
ภาษาที่ใช้เมื่อเผยแพร่รายการ หากมีการแปลพร้อมใช้งาน อินเทอร์เฟซจะแสดงในภาษานั้นด้วย
เขตเวลาของฉัน
ตัวเลือกนั้นจะกำหนดเวลาที่แสดงบนโพสต์หลังการเผยแพร่
เปลี่ยนรหัสผ่านของคุณ
เพื่อเปลี่ยนรหัสผ่านของคุณ ให้เขียนรหัสผ่านของคุณสองครั้งในช่องรหัสผ่านใหม่ และยืนยันรหัสผ่าน ความยาวของรหัสผ่านขั้นต่ำคือ 6 ตัวอักษร หากไม่มีการให้รหัสผ่านใหม่ รหัสผ่านปัจจุบันจะถูกเก็บไว้(หากคุณอัปเดตช่องนี้ คุณต้องป้อนรหัสผ่านปัจจุบันของคุณในช่องที่เหมาะสมที่ด้านล่างของแบบฟอร์ม)
การยืนยันตัวตนสองปัจจัย
การยืนยันตัวตนแบบสองปัจจัยใช้การยืนยันตัวตนด้วยรหัสผ่านแบบใช้ครั้งเดียว (One Time Password) และใช้เป็นการเพิ่มเติมจากการยืนยันตัวตนแบบมาตรฐาน (การเข้าสู่ระบบด้วยชื่อผู้ใช้และรหัสผ่าน) เท่านั้น ในการเปิดใช้งานคุณสมบัตินี้ ให้สแกน QR code ด้วยแอปพลิเคชันสำหรับการยืนยันตัวตนหรือใช้ส่วนขยายของเบราว์เซอร์ จากนั้นป้อนรหัสยืนยัน เมื่อเข้าสู่ระบบ ให้ป้อนรหัสที่ได้รับจากแอปพลิเคชันเมื่อมีการแจ้งเตือน
กุญแจการตรวจสอบตัวตน
กุญแจการตรวจสอบตัวตนถูกใช้แทนการตรวจสอบตัวตนแบบมาตรฐานผ่านชื่อผู้ใช้และรหัสผ่าน สามารถเป็นกุญแจทางกายภาพ, ปลั๊กอินของเบราว์เซอร์, หรือผู้จัดการรหัสผ่านได้ ต้องเปิดใช้งาน Javascript และ API ด้านการรับรองความถูกต้องของเบราว์เซอร์เพื่อใช้คุณสมบัตินี้ สามารถลงทะเบียนกุญแจได้หลายตัว
แอปพลิเคชันการตรวจสอบสิทธิ์
หากมีการกำหนดค่าไว้ แอปพลิเคชันของบุคคลที่สามต่าง ๆ จะสามารถใช้งานได้ โดยจะใช้แทนการยืนยันตัวตนแบบมาตรฐานด้วยชื่อผู้ใช้และรหัสผ่าน คุณสามารถอนุญาตหรือเพิกถอนสิทธิ์การเข้าถึงแอปพลิเคชันเหล่านี้ได้ หากคุณมีบัญชีบนแพลตฟอร์มของแอปพลิเคชันนั้น ๆ จากนั้นคุณสามารถเข้าสู่ระบบโดยใช้บัญชีเหล่านั้นได้

ตัวเลือกของฉัน

อินเตอร์เฟซ
เปิดใช้งานอัปโหลดเวอร์ชันปรับปรุงในตัวจัดการสื่อ
อนุญาตให้เปิดใช้งานอินเตอร์เฟซขั้นสูงในผู้จัดการสื่อ ซึ่งช่วยให้สามารถอัปโหลดไฟล์หลายไฟล์พร้อมกันได้ (หมายเหตุ: เว็บเบราว์เซอร์ของคุณต้องรองรับ Javascript จึงจะสามารถใช้คุณสมบัตินี้ได้)
ปิดการใช้งานการลากและวางที่ขับเคลื่อนด้วย Javascript สำหรับการจัดเรียงรายการ
อนุญาตให้ปิดการใช้งาน "ลากและวาง" สำหรับการจัดลำดับรายการ การลากและวางจะถูกแทนที่ด้วยช่องตัวเลขซึ่งจะให้คุณป้อนตำแหน่งของฟิลด์แทน
จำนวนรายการที่แสดงต่อหน้าในตัวจัดการสื่อ
กำหนดจำนวนสื่อที่จะแสดงในตัวจัดการสื่อ (ไม่รวมโฟลเดอร์เมื่อนับรายการ)
อย่าใช้ favicon มาตรฐาน
อนุญาตให้ปิดใช้งาน favicon เริ่มต้นในแถบที่อยู่ของเว็บเบราว์เซอร์
ฉบับ
รูปแบบที่ต้องการ
เลือกไวยากรณ์ของรายการ. Wiki คือไวยากรณ์ที่ถูกทำให้ง่ายขึ้น และจะถูกแปลงเป็น HTML ที่ถูกต้อง. หากคุณไม่มีความเข้าใจอย่างสมบูรณ์เกี่ยวกับ HTML เราขอแนะนำให้คุณเลือกไวยากรณ์ Wiki. ดูเอกสารอ้างอิงไวยากรณ์ Wiki สำหรับข้อมูลเพิ่มเติม.
สถานะการเข้าใช้งานเริ่มต้น
สถานะเริ่มต้นสำหรับรายการใหม่สามารถตั้งค่าเป็น:
  • รอดำเนินการ:สถานะการตีพิมพ์ยังไม่ได้รับการตัดสินใจ
  • กำหนดเวลา:รายการจะถูกตั้งค่าออนไลน์ในเวลาและวันที่ที่ระบุไว้ในช่องวันที่เผยแพร่
  • ไม่ได้เผยแพร่:ข้อมูลที่ไม่ได้เชื่อมต่ออินเทอร์เน็ต
  • เผยแพร่:การลงทะเบียนออนไลน์
ความสูงของช่องแก้ไขข้อมูลการเข้า
ตั้งค่าความสูงของช่องข้อมูลฉบับของรายการ โดยค่าเริ่มต้น พารามิเตอร์นี้จะถูกตั้งค่าเป็น 24
เปิดใช้งานโหมด WYSIWYG
อนุญาตให้ใช้ตัวแก้ไขแบบเห็นผลตามจริง (WYSIWYG: What You See Is What You Get) สำหรับช่องคำอธิบายของรายการ หน้า และหมวดหมู่
ตัวเลือกอื่น ๆ
รูปแบบรายการแท็ก
คุณสามารถเลือกได้ว่าจะดูขณะแก้ไขรายการหรือไม่ว่าต้องการดูแท็กทั้งหมดที่ใช้ (แบบขยาย) หรือเฉพาะแท็กที่ใช้บ่อยที่สุด (แบบย่อ)
การเน้นไวยากรณ์ในตัวแก้ไขธีม
เปิดใช้งานการเน้นไวยากรณ์เมื่อใช้ตัวแก้ไขธีม (ไฟล์ HTML และ CSS)

แดชบอร์ดของฉัน

รายการโปรดของฉัน
โซนนี้แสดงรายการโปรดที่เลือกไว้บนแดชบอร์ดของบล็อก คุณสามารถจัดเรียงลำดับใหม่ได้ด้วยเมาส์หากเปิดใช้งาน Javascript ในเบราว์เซอร์ของคุณ มิฉะนั้น จะมีช่องให้คุณระบุหมายเลขลำดับ เมื่อคุณทำเสร็จแล้ว อย่าลืมบันทึกการเปลี่ยนแปลงโดยกดปุ่มบันทึกคำสั่ง
คุณยังสามารถลบรายการโปรดได้โดยเลือกและกดปุ่มลบรายการโปรดที่เลือก
กำหนดเป็นรายการโปรดเริ่มต้น (ผู้ดูแลระบบระดับสูงสุด)
เปิดใช้งานเพื่อตั้งค่าลิสต์รายการโปรดที่ตั้งไว้ในปัจจุบันเป็นลิสต์รายการโปรดเริ่มต้นสำหรับผู้ใช้ทุกคน. ลิสต์นี้จะถูกเสนอให้กับผู้ใช้หากพวกเขายังไม่ได้ตั้งค่ารายการโปรดของตัวเองไว้.
รายการโปรดอื่น ๆ ที่มีให้บริการ
ปลั๊กอินทั้งหมดที่สามารถตั้งค่าให้เป็นรายการโปรดจะแสดงอยู่ที่นี่ ให้ทำเครื่องหมายในช่องที่สอดคล้องกันและเพิ่มไปยังแดชบอร์ดโดยกดปุ่มเพิ่มไปยังรายการโปรดของฉัน
แสดงรายการโปรดไว้ด้านบนของเมนู
หากเปิดใช้งาน จะแสดงรายการโปรดที่ด้านบนของเมนู
โมดูลแดชบอร์ด
ตัวเลือกที่ทำในโซนนี้จะปรากฎบนแดชบอร์ดของบล็อกคุณ
แสดงลิงก์เอกสาร
เปิดใช้งานการแสดงลิงก์เอกสารบนหน้าแรกแดชบอร์ด
แสดงข่าว Dotclear
เปิดใช้งานการแสดงข่าว Dotclear บนหน้าแรกแดชบอร์ด
แสดงแบบฟอร์มกรอกข้อมูลด่วน
เปิดใช้งานการแสดงแบบฟอร์ม "การป้อนข้อมูลอย่างรวดเร็ว" บนหน้าแรกแดชบอร์ด
แสดงจำนวนงานที่ค้างชำระบนไอคอนแดชบอร์ดการบำรุงรักษา
สามารถแสดงจำนวนงานที่ค้างอยู่ (คืองานที่ควรได้รับการดำเนินการ) ภายใต้ไอคอนการบำรุงรักษา ไอคอนการบำรุงรักษาต้องถูกตั้งค่าไว้ในรายการโปรดของคุณเพื่อใช้คุณสมบัตินี้
แสดงรายการงานที่ค้างชำระบนรายการแดชบอร์ด
เปิดใช้งานการแสดงรายการงานที่ค้างชำระ (คือ งานที่ควรดำเนินการ) ในส่วนเฉพาะบนหน้าแรกของแดชบอร์ด
