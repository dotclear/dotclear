แทร็กแบ็ก

หากต้องการทำแทร็กแบ็ก ให้คลิกที่ลิงก์Ping blogsหากคุณไม่เห็นลิงก์นี้ ให้ตรวจสอบให้แน่ใจว่าสถานะของรายการของคุณถูกตั้งค่าเป็นเผยแพร่แล้ว

URLs สำหรับ ping
วางที่นี่ URL ของ trackback ที่คุณพบในโพสต์ที่คุณต้องการ ping
ข้อความตัวอย่างที่จะส่ง
โดยค่าเริ่มต้น ช่องนี้จะแสดงข้อความเริ่มต้นของรายการของคุณ นี่คือสิ่งที่จะถูกส่งไปยังบล็อกที่คุณกำลังส่ง Ping พร้อมด้วยลิงก์ไปยังโพสต์ของคุณ คุณสามารถแก้ไขข้อความสรุปได้โดยแก้ไขช่องนี้
ค้นหาและส่ง ping ไปยัง URL โดยอัตโนมัติ
หากการเชื่อมโยงของคุณไปยังโพสต์ที่เฉพาะเจาะจง และหากบล็อกที่โฮสต์โพสต์เหล่านั้นตั้งค่าให้อนุญาตไว้ เครื่องมือนี้จะค้นหา URL ที่ต้องส่ง ping ให้คุณ
