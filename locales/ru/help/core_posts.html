Фильтр списка записей

Автор
Фильтрация записей по автору.
Категория
Фильтры записей по категориям.
Статус
Фильтрации по статусу:
  • В ожидании: Ожидающие записи.
  • Запланировано: Запланированные записи.
  • Не опубликовано: записей в автономном режиме.
  • Опубликовано: онлайн записей.
Выбрано
Фильтрует записи, которые имеют Выбранный статус или нет.
Вложения
Фильтры записи основанные на наличии вложений или нет.
Месяц
Фильтрует записи за данный месяц данного года.
Язык
Фильтрация записей на основе языка, установленного в поле Язык записи.
Сортировать по
Позволяет заказывать результат по дате, названию, категории, автору, статусу публикации, выбранному статусу, количеству комментариев или количеству обратных ссылок.
Сортировка
Позволяет сортировать результаты по возрастанию или убыванию.
Записи на странице
Количество записей для отображения на странице.

Пакетные действия

Можно совершать несколько действий на несколько статей. Возможные действия зависят от прав пользователя:

  • Опубликовать: Опубликовать выбранные записи онлайн.
  • Снять публикацию: Потяните выбранные записи оффлайн.
  • Расписание: Расписание выбранных записей, которые будут заданы онлайн на дату публикации.
  • Отметить в ожидании: отметить выбранные записи в ожидании.
  • Отметить как выбранные/невыбранные: Установить выбранные записи в качестве выделенных/невыбранных./li>
  • Изменить категорию: Изменить категорию выбранных записей.
  • Изменить язык: Изменить язык выбранных записей.
  • Автор изменения: Изменить автора выбранных записей, введя новый авторский ID.
  • Удалить: Удалить выбранные записи (эту операцию нельзя отменить).
  • Добавить/Удалить теги: Добавить или удалить теги из выбранных записей.

Вышеприведенный список может содержать другие элементы, если дополнительные действия были добавлены плагинами, активными в системе.
