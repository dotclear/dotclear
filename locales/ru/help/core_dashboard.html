Персонализация панели инструментов

Избранное
Большие иконки в середине страницы позволяют быстро получить доступ к вашим любимым страницам администрирования.
Модули панели инструментов
Модули, отображаемые в избранном устанавливаются по умолчанию или добавляются плагинами.

You can personalize the display of favorites and modules by going in the My dashboard tab in the Preferences page.

Главное меню

Вы можете отобразить или скрыть это меню в любое время, нажав на его правой стороне (или активировав Скрыть главное меню ссылка).

The first part of the menu lists all the favorite pages of the user. You can select to display this list or not in the My dashboard tab in the Preferences page.

Остальные меню содержат администраторские страницы. Поделены на три раздела:

  • Блог для всего относительно содержимого и параметров текущего блога.
  • Системные настройки для управления глобальной установкой.
  • Плагины для расширений, затрагивающих всю систему.
