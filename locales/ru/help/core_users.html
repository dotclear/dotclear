На этой странице перечислены все пользователи системы.

Чтобы добавить (или изменить) разрешения пользователей, необходимо ввести пароль в поле Ваш пароль.

Фильтр списка пользователей

Искать
Поиск основан на ID пользователя. * подстановочный знак может быть использован для поиска только части ID пользователя.
Сортировать по
Список пользователей можно упорядочить по их ID, их Фамилия, Имя, их Отображаемое Имя (ник) или Количество записей они создали.
Сортировка
Показывает метод сортировки, который будет применяться к списку результатов (По возрастанию или по убыванию).
Пользователей на странице
Устанавливает количество отображаемых пользователей на каждой странице списка результатов.

Пакетные действия пользователей

Можно сделать несколько действий на нескольких пользователей одновременно.

  • Установите разрешения: Установите права доступа для одного или нескольких блогов (блогов).
  • Удалить: Удалить выбранного пользователя(ов) из системы. Пользователь, который создал записи не может быть удален.

Если вы решите установить права пользователя, вы будете перенаправлены на новую страницу с блогами, доступными в вашей системе. Проверьте блоги, которые вы хотите установить разрешения, и нажмите Установить разрешения.

Права пользователей

Для каждого блога, вы можете установить следующие права:

  • Администратор: Может администрировать блог целиком.
  • Управление собственными записями и комментариями Пользователь может создавать записи и управлять комментариями к записям, которые он создал.
  • Опубликовать записи и комментарии: Пользователь может изменять статус публикации своих записей и свои комментарии.
  • Удалить записи и комментарии: Пользователь может удалить свои собственные записи, а также комментарии его записей.
  • Управление всеми записями и комментариями: Пользователь может управлять всеми записями этого блога, независимо от того, автор он или нет.
  • Управление категориями: Пользователь может создавать, изменять и удалять категории.
  • Управлять своими материалами: Пользователь может добавлять медиафайлы и модифицировать или удалять свои собственные медиафайлы.
  • Управление всеми элементами медиа: Пользователь может управлять элементами мультимедиа от любого пользователя.
  • Управление страницами: Пользователь может добавлять и изменять страницы.
  • Управление блогом: Пользователь может управлять блогом текущего блога.

Статус Супер администратор устанавливается в профиле пользователя.
