Installierte Plugins

Jede Zeile beginnt mit dem Plugin-Logo, das Sie im Menü (auf der linken Seite Ihres Bildschirms) finden.

Ein Dotclear-Logo neben den Aktionsschaltflächen zeigt an, dass das Plugin direkt von Dotclear stammt und eine seiner Komponenten ist.

Aktivierte Plugins

Diese Liste zeigt Informationen über aktivierte Plugins an: Name, Version, Kurzbeschreibung. Es sind zwei Aktionen möglich: Deaktivieren (temporärer Vorgang) oder Löschen (permanenter Vorgang).

Deaktivierte Plugins

Diese Liste zeigt die Plugins an, die vom Benutzer deaktiviert wurden. Für jedes Plugin sind zwei Aktionen möglich: Aktivieren Sie oder Löschen Sie (permanenter Vorgang).

Plugins hinzufügen

Diese Seite ermöglicht es Ihnen, das DotAddict-Repository zu durchsuchen und ein Plugin direkt daraus zu installieren.

Es stehen zwei Suchmethoden zur Verfügung:

  • Geben Sie den Namen des Plugins in das Feld Search in repository ein,
  • Durchsuchen Sie den Index

Wenn Sie im Index ein Plugin ausklappen, indem Sie auf den Pfeil neben dem Namen klicken, gelangen Sie auf die DotAddict-Seite oder zu einer Support-Website, falls der Plugin-Autor eine solche angegeben hat.

Manuelle Installation oder Aktualisierung

Sie können Ihre eigenen Plugins oder Plugins, die nicht auf DotAddict verfügbar sind, über diese Seite installieren.

Es gibt zwei Möglichkeiten, sie zu installieren:

  • Laden Sie eine Zip-Datei von Ihrem Computer hoch. Klicken Sie auf Durchsuchen , um die Datei auszuwählen, geben Sie Ihr Passwort ein und drücken Sie Hochladen.
  • Laden Sie eine Zip-Datei direkt aus dem Internet herunter, indem Sie die URL der Datei eingeben, dann Ihr Passwort eingeben und auf Downloaddrücken.
