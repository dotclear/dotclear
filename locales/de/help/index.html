Erkundung der Schnittstelle

Kontextbezogene Hilfe

Zusätzlich zu den globalen Hilfeseiten enthalten mehrere Verwaltungsseiten eine Online-Hilfe. Wenn JavaScript in Ihrem Browser aktiviert ist, können Sie die Online-Hilfe oben rechts auf der Seite anzeigen, indem Sie auf den Link Need Help? Wenn JavaScript deaktiviert ist, wird die Hilfe unten auf der Seite angezeigt.

Auf jeder Verwaltungsseite finden Sie den gleichen oberen Bereich, das Hauptmenü und die Fußzeile. (Die Positionen dieser Elemente sind hier für die Anzeige auf einem Computer dargestellt; auf einem mobilen Gerät oder Tablet variiert ihre Platzierung je nach verfügbarem Platz).

Banner oben

Der Abschnitt oben auf jeder Seite besteht aus drei Elementen:

Logo

Standardmäßig ist dieses Logo das Dotclear-Logo und verweist auf die Landing Page der Verwaltung. Ein Plattformadministrator kann jedoch ein anderes Logo wählen und auf eine andere Seite (in der Regel die Homepage des Dienstes) umleiten.

Blog(s)

Das erste Element nach dem Logo zeigt das aktive Blog an, d. h. das Blog, an dem Sie gerade arbeiten. Wenn Ihre Installation aus mehreren Blogs besteht, ist diese Information in einem Dropdown-Menü verfügbar, mit dem Sie zu einem anderen Blog wechseln können.

Der Link nach dem Blognamen ermöglicht es Ihnen, den Blog in einem neuen Fenster oder einer neuen Registerkarte zu öffnen.

Benutzer

Ganz rechts in der oberen Leiste finden Sie den Link Mein Dashboard, der Sie zurück zur Startseite des Blogs führt.

Der nächste Link, Meine Einstellungen, führt Sie zu der Seite, auf der Sie Ihre persönlichen Daten bearbeiten, Ihr Dashboard konfigurieren und Ihre Schnittstelle und Bearbeitungsoptionen auswählen können.

Überraschenderweise können Sie sich über den Link Abmelden [Ihre Anmeldung]… abmelden, ja, wirklich. (Bei Dotclear sind wir wirklich gut in Ergonomie.)

Hauptmenü

Das Hauptmenü befindet sich auf der linken Seite des Bildschirms. Es kann durch Anklicken des rechten Randes eingeklappt (und durch Anklicken desselben Randes ausgeklappt) werden.

Sie führt zu den verschiedenen Verwaltungsseiten, die Ihren Berechtigungen für den aktiven Blog entsprechen. Sie besteht aus vier Abschnitten 

  • Meine Favoriten listet die Seiten auf, die Sie auf der Registerkarte Mein Dashboard unter Meine Einstellungen als Favoriten markiert haben. Wenn Sie noch keine Favoriten ausgewählt haben, zeigt diese Liste die Standardeinstellung von Dotclear oder die des Plattformadministrators. Wenn Sie diesen Block nicht anzeigen möchten, deaktivieren Sie das entsprechende Kontrollkästchen in Ihren Dashboard-Optionen.
  • Blog listet alle Seiten auf, die es Ihnen ermöglichen, das aktuelle Blog zu aktualisieren und zu konfigurieren, sei es die Bearbeitung eines Beitrags, die Verwaltung von Medien, die Auswahl des Erscheinungsbilds des Blogs oder die Anpassung anderer Einstellungen.
    Wenn Sie keine Administratorrechte für das aktuelle Blog haben, gibt es keinen weiteren Block unterhalb dieses Blocks im Hauptmenü.
  • Die Systemeinstellungen bieten Zugriff auf Seiten zur Verwaltung der Dotclear-Installation: Blog-Verwaltung, Benutzer und verfügbare Sprachen, Installation von Erweiterungen und Systemeinstellungen. Wenn Sie über Berechtigungen für mehrere Blogs verfügen, aber nicht den Status eines Superadministrators haben, besteht sie nur aus dem Element Blogs.
  • Unter Plugins können Sie die Einstellungen für installierte Plugins konfigurieren. Wenn die Einstellungen eines Plugins jeweils nur einen Blog betreffen, ist seine Verwaltungsseite im Abschnitt Blog verfügbar; wenn die Einstellungen jedoch alle Seiten der Installation betreffen, erscheinen sie hier und sind nur für Benutzer mit Super-Admin-Rechten zugänglich.

Fußzeile

Eine kostenlose Version von Dotclear ist für jeden Benutzer erhältlich, der das Geheimnis der Fußzeile entdeckt.

Hilfe für absolute Anfänger

Wenn Sie mit Blogs nicht vertraut sind, empfehlen wir Ihnen, unsere Dokumentation zu konsultieren (das vollständige Inhaltsverzeichnis finden Sie auf der Registerkarte Dokumentation auf dieser Seite).

Das Dotclear-Forum ist ein Ort der gegenseitigen Unterstützung, an dem es keine dummen Fragen gibt. Fühlen Sie sich frei, sich zu beteiligen, wenn Sie Hilfe benötigen, und warum nicht, helfen Sie anderen im Gegenzug.

Wir bemühen uns, ein Vokabular zu verwenden, das auch ohne Vorkenntnisse verständlich ist. Sollten wir dennoch versehentlich einen Fachbegriff verwendet haben, so finden Sie wahrscheinlich eine Erklärung im Glossar unserer Online-Dokumentation.

Zugänglichkeit und Benutzerfreundlichkeit

Die Dotclear-Benutzeroberfläche erfüllt fast alle WCAG 2.0-Kriterien; ein ausführlicher Bericht wird demnächst auf dem Projektblog veröffentlicht. Letztendlich möchten wir auch die ATAG-2.0-Kriterien vollständig erfüllen.

Barrierefreiheit und Benutzerfreundlichkeit sind für das Dotclear-Team zentrale Anliegen. Wir führen regelmäßig Analysen mit Hilfe von Experten durch und implementieren kontinuierlich Korrekturen und Verbesserungen.

Sie können uns helfen, indem Sie Tickets eröffnen oder im Forum teilnehmen. Sie können auch an Benutzertests teilnehmen oder unserem Team beitreten!
