Auf dieser Seite werden alle Benutzer des Systems aufgelistet.

Zum Hinzufügen (oder Ändern) von Benutzerberechtigungen ist Ihr Passwort im Feld Ihr Passwort erforderlich.

Filter für die Benutzerliste

Suche
Die Suche basiert auf der Benutzer-ID. Der Platzhalter * kann verwendet werden, um nur nach einem Teil der Benutzerkennung zu suchen.
Bestellen bei
Die Benutzerliste kann nach ihrer ID, ihrem Nachnamen, ihrem Vornamen, ihrem Anzeigenamen (Spitzname) oder der Anzahl der von ihnen erstellten Einträge geordnet werden.
Sortieren
Gibt die Sortiermethode an, die auf die Ergebnisliste angewendet wird (Aufsteigend oder Absteigend).
Benutzer pro Seite
Legt die Anzahl der Benutzer fest, die auf jeder Seite der Ergebnisliste angezeigt werden.

Benutzer Batch-Aktionen

Es ist möglich, einen Stapel von Aktionen für mehrere Benutzer auf einmal durchzuführen.

  • Berechtigungen festlegen: Legen Sie die Berechtigungen für ein oder mehrere Blog(s) fest.
  • Löschen: Entfernen Sie den/die ausgewählten Benutzer aus dem System. Ein Benutzer, der Einträge erstellt hat, kann nicht gelöscht werden.

Wenn Sie sich entscheiden, die Benutzerrechte festzulegen, werden Sie zu einer neuen Seite weitergeleitet, auf der die in Ihrem System verfügbaren Blogs aufgelistet sind. Markieren Sie die Blogs, für die Sie die Berechtigungen festlegen möchten, und klicken Sie auf Berechtigungen festlegen.

Benutzerberechtigungen

Für jedes Blog können Sie die folgenden Berechtigungen festlegen:

  • Verwalter: Kann den Blog vollständig verwalten.
  • Verwalten Sie Ihre eigenen Einträge und Kommentare: Der Benutzer kann Einträge erstellen und die Kommentare zu den von ihm erstellten Einträgen verwalten.
  • Einträge und Kommentare veröffentlichen: Der Benutzer kann den Veröffentlichungsstatus seiner eigenen Einträge und der Kommentare zu seinen Einträgen ändern.
  • Einträge und Kommentare löschen: Der Benutzer kann sowohl seine eigenen Einträge als auch die Kommentare zu seinen Einträgen löschen.
  • Verwalten Sie alle Einträge und Kommentare: Der Benutzer kann alle Einträge in diesem Blog verwalten, unabhängig davon, ob er der Autor ist oder nicht.
  • Verwalten von Kategorien: Der Benutzer kann Kategorien erstellen, ändern und löschen.
  • Verwalten Sie Ihre eigenen Medienelemente: Der Benutzer kann Medienelemente hinzufügen und seine eigenen Medien ändern oder löschen.
  • Verwalten Sie alle Medienelemente: Der Benutzer kann die Medienelemente eines beliebigen Benutzers verwalten.
  • Verwalten von Seiten: Der Benutzer kann Seiten hinzufügen und ändern.
  • Blogroll verwalten: Der Benutzer kann die Blogroll des aktuellen Blogs verwalten.

Der Status Superadministrator wird im Benutzerprofil festgelegt.
