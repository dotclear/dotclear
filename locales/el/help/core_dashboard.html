Κεφαλίδα

Αν υπάρχουν πολλά blogs μπορείτε να έχετε πρόσβαση σε αυτό το παράδειγμα, μια αναπτυσσόμενη λίστα σας επιτρέπει να επιλέξετε αυτό που θέλετε να εργαστείτε. Αν υπάρχει μόνο ένα blog, εμφανίζει απλά το όνομά του.

Ο ακόλουθος σύνδεσμος επιτρέπει να πάτε στο τρέχον blog (ανοίγει σε μια νέα καρτέλα).

Οι άλλοι σύνδεσμοι σας επιτρέπουν να επιστρέψετε σε αυτόν τον πίνακα ελέγχου, να αποκτήσετε πρόσβαση στις προτιμήσεις σας και να αποσυνδεθείτε.

Εξατομίκευση του ταμπλό

Αγαπημένα
Τα μεγάλα εικονίδια στη μέση της σελίδας σας επιτρέπουν να έχετε γρήγορη πρόσβαση στις αγαπημένες σας σελίδες διαχείρισης.
Πρόσθετα ταμπλό
Τα πρόσθετα που εμφανίζονται στα αγαπημένα εγκαθίστανται από προεπιλογή ή προστίθενται από πρόσθετα.

You can personalize the display of favorites and modules by going in the My dashboard tab in the Preferences page.

Το κύριο μενού

Μπορείτε να εμφανίσετε ή να αποκρύψετε αυτό το μενού οποιαδήποτε στιγμή κάνοντας κλικ στη δεξιά πλευρά του (ή ενεργοποιώντας το Απόκρυψη του κύριου μενού link).

The first part of the menu lists all the favorite pages of the user. You can select to display this list or not in the My dashboard tab in the Preferences page.

Το υπόλοιπο μενού εμφανίζει τις σελίδες διαχείρισης. Χωρίζεται σε τρεις ενότητες:

  • Blog για τα πάντα σχετικά με το τρέχον περιεχόμενο και τις παραμέτρους ιστολογίου.
  • Ρυθμίσεις συστήματος για τη διαχείριση της καθολικής εγκατάστασης.
  • Πρόσθετα για επεκτάσεις που επηρεάζουν ολόκληρο το σύστημα.
