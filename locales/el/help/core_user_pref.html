Το προφίλ μου

Επώνυμο, Όνομα
Εάν το εμφανιζόμενο όνομα είναι κενό, το όνομα του συγγραφέα θα εμφανίζεται ως το όνομα και το επώνυμό του όπως ορίζονται εδώ.
Εμφανιζόμενο όνομα
Εδώ μπορείτε να επιλέξετε ένα ψευδώνυμο που θα εμφανίζεται στις αναρτήσεις σας αντί για το επίθετο και το όνομά σας.
E-mail
Διεύθυνση ηλεκτρονικού ταχυδρομείου που χρησιμοποιείται όταν ζητάτε νέο κωδικό πρόσβασης.
Θα αποστέλλονται ειδοποιήσεις νέων σχολίων και σε αυτήν τη διεύθυνση.(Εάν ενημερώσετε αυτό το πεδίο, πρέπει να εισαγάγετε τον τρέχοντα κωδικό πρόσβασής σας στο κατάλληλο πεδίο στο κάτω μέρος της φόρμας)
URL
Ο ιστότοπος του χρήστη. Εάν οριστεί, το όνομα του συγγραφέα θα εμφανίζεται ως σύνδεσμος προς τον εν λόγω ιστότοπο.
Γλώσσα για το περιβάλλον εργασίας μου
Η γλώσσα που χρησιμοποιείται κατά τη δημοσίευση καταχωρήσεων. Εάν η μετάφραση είναι διαθέσιμη, η διεπαφή θα εμφανίζεται επίσης σε αυτήν τη γλώσσα.
Η ζώνη ώρας μου
Αυτή η επιλογή θα καθορίσει την ώρα που θα εμφανίζεται στη δημοσίευση της δημοσίευσης.
Αλλάξτε τον κωδικό σας
Για να αλλάξετε τον κωδικό πρόσβασής σας, σημειώστε δύο φορές τον κωδικό πρόσβασής σας στα πεδία Νέος κωδικός και Επιβεβαίωση κωδικού πρόσβασης. Το ελάχιστο μήκος κωδικού πρόσβασης είναι 6 χαρακτήρες. Εάν δεν παρέχεται νέος κωδικός πρόσβασης, διατηρείται ο τρέχων. (Εάν ενημερώσετε αυτό το πεδίο, πρέπει να εισαγάγετε τον τρέχοντα κωδικό πρόσβασής σας στο κατάλληλο πεδίο στο κάτω μέρος της φόρμας)

Οι επιλογές μου

Διεπαφή
Ενεργοποίηση βελτιωμένου uploader στο πρόγραμμα διαχείρισης πολυμέσων
Επιτρέπει την ενεργοποίηση της προηγμένης διεπαφής στη διαχείριση πολυμέσων που επιτρέπει τη μεταφόρτωση πολλών αρχείων ταυτόχρονα (Σημείωση: Το πρόγραμμα περιήγησής σας στο Web πρέπει να υποστηρίζει Javascript για να μπορείτε να χρησιμοποιήσετε αυτήν τη δυνατότητα).
Απενεργοποιήστε το Javascript powered drag and drop για την παραγγελία στοιχείων
Επιτρέπει την απενεργοποίηση του "drag and drop" για την παραγγελία αντικειμένων. Η μεταφορά και απόθεση θα αντικατασταθεί από αριθμητικά πεδία που θα σας επιτρέψουν να εισαγάγετε τη θέση των πεδίων.
Αριθμός στοιχείων που εμφανίζονται ανά σελίδα στο διαχειριστή πολυμέσων
Καθορίζει τον αριθμό των μέσων που θα εμφανίζονται στη διαχείριση πολυμέσων (οι φάκελοι δεν λαμβάνονται υπόψη κατά την καταμέτρηση στοιχείων).
Μην χρησιμοποιήσετε το τυπικό favicon
Επιτρέπει την απενεργοποίηση του προεπιλεγμένου εικονιδίου αγαπημένων στη γραμμή διευθύνσεων του προγράμματος περιήγησης.
Έκδοση
Προτιμώμενη μορφή
Επιλέξτε τη σύνταξη της καταχώρισης. Το Wiki είναι μια απλοποιημένη σύνταξη και θα μετατραπεί σε έγκυρο HTML. Αν δεν έχετε τέλεια κατανόηση του HTML, σας συμβουλεύουμε να επιλέξετε τη σύνταξη του Wiki. Δείτε την αναφορά σύνταξης Wiki για περισσότερες πληροφορίες.
Προεπιλεγμένη κατάσταση καταχώρησης
Η προεπιλεγμένη κατάσταση για τις νέες καταχωρήσεις μπορεί να οριστεί σε:
  • Αναμονή: Η κατάσταση δημοσίευσης δεν έχει ακόμα αποφασιστεί.
  • Προγραμματισμός: Η καταχώρηση θα οριστεί online την ώρα και την ημερομηνία που παρέχεται στο πεδίο Δημοσιεύθηκε στο.
  • Αδημοσίευτο: Καταχώρηση χωρίς σύνδεση.
  • Δημοσιεύθηκε: Ηλεκτρονική καταχώρηση.
Ύψος πεδίου επεξεργασίας εγγραφής
Ορίζει το ύψος του πεδίου έκδοσης μιας καταχώρησης. Από προεπιλογή, αυτή η παράμετρος έχει οριστεί σε 24.
Ενεργοποίηση λειτουργίας WYSIWYG
Επιτρέπει τη χρήση του οπτικού επεξεργαστή (WYSIWYG: Τι βλέπετε είναι αυτό που παίρνετε) για τα πεδία καταχωρήσεων, σελίδων και περιγραφής κατηγοριών.
Άλλες επιλογές
Μορφή λίστας ετικετών
Μπορείτε να επιλέξετε αν θέλετε να δείτε, κατά την επεξεργασία μιας καταχώρησης, all the tags used (Extended) or only the most used ones (Short).
Επισήμανση σύνταξης στον επεξεργαστή θέματος
Ενεργοποιεί την επισήμανση σύνταξης όταν χρησιμοποιείτε τον επεξεργαστή θέματος (HTML και αρχεία CSS)

Το ταμπλό μου

Τα αγαπημένα μου
Αυτή η ζώνη εμφανίζει τα επιλεγμένα αγαπημένα στον πίνακα ελέγχου του ιστολογίου. Μπορούν να ταξινομηθούν εκ νέου με το ποντίκι σας, εάν το Javascript είναι ενεργοποιημένο στο πρόγραμμα περιήγησής σας. Διαφορετικά, ένα πεδίο επιτρέπει να τους δώσει έναν αριθμό παραγγελίας. Όταν τελειώσετε, μην ξεχάσετε να αποθηκεύσετε τις αλλαγές σας πατώντας Αποθήκευση παραγγελίας.
Μπορείτε επίσης να διαγράψετε αγαπημένα επιλέγοντάς τα και πατώντας Διαγραφή επιλεγμένων αγαπημένων.
Ορισμός ως προεπιλεγμένα αγαπημένα (υπερδιαχειριστής)
Ενεργοποιήστε τον ορισμό λίστας αγαπημένων ως προεπιλεγμένη λίστα αγαπημένων για όλους τους χρήστες. Αυτή η λίστα θα προτείνεται στους χρήστες αν δεν έχουν ορίσει ακόμα τα δικά τους αγαπημένα
Άλλα διαθέσιμα αγαπημένα
Όλα τα plugins που επιτρέπουν να τοποθετούνται ως αγαπημένα diplayed εδώ. Ελέγξτε τα αντίστοιχα πλαίσια και προσθέστε τα στον πίνακα ελέγχου πατώντας Προσθήκη tp τα αγαπημένα μου.
Εμφάνιση αγαπημένων στο πάνω μέρος του μενού
Αν ενεργοποιηθεί, εμφανίζει τα αγαπημένα στο πάνω μέρος του μενού.
Πρόσθετα ταμπλό
Οι επιλογές που γίνονται σε αυτή τη ζώνη θα αντικατοπτρίζονται στο ταμπλό του ιστολογίου σας.
Εμφάνιση συνδέσμων τεκμηρίωσης
Ενεργοποιήστε την εμφάνιση συνδέσμων τεκμηρίωσης στη σελίδα προορισμού του ταμπλό.
Εμφάνιση ειδήσεων Dotclear
Ενεργοποιήστε για εμφάνιση νέων Dotclear στη σελίδα προορισμού του ταμπλό.
Εμφάνιση φόρμας γρήγορης καταχώρησης
Ενεργοποιήστε για την εμφάνιση της φόρμας "γρήγορης εισόδου" στη σελίδα προσγείωσης του ταμπλό.
Εμφάνιση μετρητή εκπρόθεσμων εργασιών στο εικονίδιο του ταμπλό συντήρησης
Επιτρέπει την εμφάνιση του αριθμού των ληξιπρόθεσμων εργασιών (δηλαδή που θα πρέπει να επεξεργαστούν) κάτω από το εικονίδιο Συντήρηση . Συντήρηση εικονίδιο πρέπει να οριστεί στα αγαπημένα σας για να χρησιμοποιήσετε αυτό το χαρακτηριστικό.
Εμφάνιση λίστας εκπρόθεσμων εργασιών στα στοιχεία του ταμπλό
Ενεργοποιήστε για να εμφανίσετε τη λίστα των εκπρόθεσμων εργασιών (δηλ. που θα πρέπει να επεξεργαστούν) σε μια ειδική ενότητα στη σελίδα προορισμού του ταμπλό.
